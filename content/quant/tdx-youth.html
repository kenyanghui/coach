通达信 青少年投资课程

面向青少年与企业二代的金融素养与投资实战培训体系
基于通达信知识体系开发的四阶递进课程

📚 4 阶 · 20 模块 · 60+ 课时 🎯 12~22 岁 / 企业接班人 🏆 通达信认证 Analyst

课程设计理念

不只是教炒股，而是建立完整的金融思维框架

1.1 为什么需要这门课

在中国，青少年金融教育存在系统性缺失。学校教微积分，但不教如何管理个人财富；学生能解方程，但看不懂资产负债表。对于企业二代而言，这个问题更加突出——他们未来要管理的是家族资产和企业财务。

本课程基于通达信知识体系，将专业的金融工具转化为适合青少年的进阶学习路径。通达信作为覆盖 90% 国内券商的行业标准软件，学习它本身就是进入金融世界的"第一张门票"。

🎯 核心目标

  • 建立正确的金钱观与投资理念
  • 掌握通达信工具的基本使用
  • 看懂 K 线、财务数据、市场信息
  • 具备独立分析判断的初步能力
  • 为未来的企业财务与资产配置打基础

🧠 教学理念

  • 做中学 — 70% 实操 + 30% 理论
  • 真问题 — 用真实市场数据作为案例
  • 长周期 — 培养长期思维，拒绝暴富心态
  • 负责任 — 每一笔模拟交易都要写投资理由
  • 有产出 — 每阶段完成一份投资报告

1.2 目标学员画像

L1 · 少年金融启蒙

12~16 岁 · 16 课时 · 核心目标：建立兴趣、正确金钱观、通达信初体验

2.1 L1 课程大纲

L1-01：钱是什么？

讲授 2 课时

核心问题：钱从哪里来？为什么一张纸可以买到东西？通货膨胀是什么？

活动：小组讨论"如果你有 100 万会怎么做"，观看通胀模拟动画。

通达信任务：安装通达信，认识界面。看看自己知道的公司的股票（茅台、腾讯、阿里）。

L1-02：储蓄 vs 投资

讲授 + 实操 2 课时

核心概念：利息、复利、风险与收益的关系。

活动：复利计算器实验——每个月存 100 元，到 60 岁是多少？

通达信任务：打开基金板块，看看货币基金、债券基金、股票基金的不同。

L1-03：什么是股票？

讲授 + 实操 2 课时

核心概念：股票是公司的所有权凭证。股东是什么？分红是什么？

活动："开一家奶茶店"模拟——分股份、分红、股价波动。

通达信任务：搜索 10 家知名公司，添加到自选股。看看它们的股价和公司名。

L1-04：认识 K 线

讲授 + 实操 2 课时

核心概念：开盘价、收盘价、最高价、最低价。阳线阴线。为什么 K 线是"蜡烛"？

活动：每组发一张股票的月度股价数据，手动画 K 线图。

通达信任务：用 F5 在分时和 K 线之间切换。用 ←→ 查看历史。

L1-05：什么是公司？

讲授 + 实操 2 课时

核心概念：上市公司做什么？营收、利润、员工——最简单的公司画像。

活动：选一家熟悉的公司（如苹果/麦当劳/伊利），上网查它的业务。

通达信任务：按 F10 看公司资料——公司概况、主营业务。找到"它靠什么赚钱"。

L1-06：市场是什么？

讲授 + 实操 2 课时

核心概念：上证指数、深证成指是什么？为什么说"大盘涨了"？

活动：模拟市场游戏——每组代表一个公司，全班交易形成股价。

通达信任务：输入 03+Enter 看上证指数。60+Enter 看涨幅排名。

L1-07：我的投资组合

实操 + 项目 2 课时

任务：虚拟 10 万元，选择 3~5 只股票构建投资组合。

要求：每选一只股票必须写出"为什么选它"（至少 50 字）。

通达信任务：建立自选板块"我的组合"，设置价格预警。

L1-08：投资报告会

项目 + 考核 2 课时

产出：3 分钟演讲——"我的投资组合和理由"。

考核标准：逻辑清晰（40%）、表达流畅（30%）、有反思（30%）。不考核收益率。

L1 通关条件：完成报告 + 通达信基础操作测试（开机、搜股票、看 K 线、看 F10）。

L1 教学要点：绝对不鼓励真实投资。所有交易均为模拟。重点是建立正确的认知框架，而非追求收益率。本金永久损失的风险意识从第一课开始强调。

L2 · 投资分析入门

14~18 岁 · 24 课时 · 核心目标：掌握通达信分析工具，看懂技术与基本面

3.1 L2 课程大纲

L2-01：通达信高手之路

实操 3 课时

内容：快捷键强化训练、界面自定义、自选股管理、板块分类。

通达信知识体系对应：第一篇 1.4~1.6（界面布局、基本操作、快捷键大全）

练习：快捷键计时赛——老师报功能，学生用最快速度操作。

L2-02：K 线语言

讲授 + 实操 3 课时

内容：K 线组合（十字星、启明星、黄昏星、红三兵、三只乌鸦）。

通达信知识体系对应：第一篇 1.7（K线图）

活动："K 线猜谜"——只看 K 线形态猜后续涨跌。然后揭晓真实走势。

L2-03：技术指标入门

讲授 + 实操 3 课时

内容：MA（均线）、MACD、KDJ、RSI、VOL 的含义和基本用法。

通达信知识体系对应：第一篇 1.8（技术指标） + 第二篇 2.2（通达信指标评分9.8）

练习：每人在通达信调出 MACD + KDJ，在历史 K 线上找"金叉"和"死叉"的例子。

重点强调：指标不是预测未来的水晶球，而是描述过去的工具。没有指标能 100% 准确。

L2-04：基本面分析入门

讲授 + 实操 3 课时

内容：F10 资料详解——财务数据（营收、利润、ROE）、股东结构、分红记录。

通达信知识体系对应：第一篇 1.10（F10资料）

活动：对比两家同行业公司（如茅台 vs 五粮液），谁的财务更健康？

L2-05：条件选股实战

实操 3 课时

内容：用通达信条件选股器筛选股票。组合多个条件提高精度。

通达信知识体系对应：第一篇 1.11（条件选股）

挑战："选出市盈率 < 20、净利润增长 > 20%、MACD 金叉的股票"——看谁选出的结果最合理。

L2-06：行业分析入门

讲授 + 实操 3 课时

内容：行业分类、板块分析、热点识别。为什么有些板块涨有些跌？

通达信知识体系对应：第二篇全（行业分析）

练习：用 94+Enter 看今日板块涨幅排名。选一个领涨板块，找出龙头股。

L2-07：模拟交易大赛

实操 + 项目 3 课时

规则：虚拟 50 万，模拟交易 4 周。每日收盘后记录持仓和操作理由。

要求：每笔交易必须写分析逻辑（技术面理由 + 基本面理由）。

复盘：每周一次小组复盘——你的交易有逻辑吗？哪里做对了？哪里错了？

L2-08：投资分析报告 + 认证考试

项目 + 考核 3 课时

产出：一份完整的个股分析报告（2000字+），包含：行业分析、公司基本面、技术面分析、投资结论。

考核：通达信操作技能考试（30分钟）、分析报告答辩（每人15分钟）。

L2 通关条件：分析报告 ≥ 70 分 + 操作考试 ≥ 80 分。

L2 教学要点：技术面和基本面并重。不鼓励"炒短线"，每笔模拟交易持仓不少于 3 天。强调"逻辑比结果重要"——错得清楚比赚得糊涂更有价值。

L3 · 实战交易训练

16~22 岁 · 24 课时 · 核心目标：深入理解市场运行规律，建立交易系统

4.1 L3 课程大纲

L3-01：市场微观结构

讲授 3 课时

内容：交易所运作机制、订单类型（限价单/市价单）、盘口解读、Level-2 行情。

练习：在通达信观察盘口变化——五档买卖挂单的博弈意味着什么？

L3-02：通达信公式编写入门

讲授 + 实操 3 课时

内容：通达信公式语言基础——变量、函数、条件判断。编写第一个自定义指标。

通达信知识体系对应：第二篇 2.2（通达信公式/扩展性评分9.9）

练习：编写一个 5日均线上穿 20日均线的选股公式。

L3-03：交易心理学

讲授 + 实操 3 课时

内容：恐惧与贪婪、锚定效应、确认偏误、损失厌恶——行为金融学核心概念。

活动："交易心理实验"——在模拟环境中体验大涨、大跌、踏空、卖飞的心理感受。

复盘：回看自己 L2 模拟交易记录——哪些决策是情绪驱动的？

L3-04：风险管理

讲授 + 实操 3 课时

内容：仓位管理、止损止盈设置、最大回撤、夏普比率、凯利公式入门。

通达信知识体系对应：第三篇 3.7（风控体系）

练习：设计一套自己的交易风控规则（单笔止损 X%、总仓位不超过 Y%）。

L3-05：期权期货入门

讲授 + 实操 3 课时

内容：期货是什么（远期合约、保证金、交割）、期权是什么（看涨/看跌、权利金）。

通达信知识体系对应：第三篇 3.3~3.4（量化交易、系统架构）

演示：在通达信中查看期货和期权行情，看不同类型的合约代码含义。

安全提示：期权期货有杠杆，风险远大于股票。本课仅为认知教育，不鼓励实盘操作。

L3-06：量化思维入门

讲授 + 实操 3 课时

内容：什么是量化交易？回测是什么？为什么需要规则化的交易系统？

通达信知识体系对应：第三篇 3.1~3.2（量化平台、QMT vs PTrade）

活动：手工回测一个简单规则——"5日均线上穿20日均线买入，下穿卖出"，用 Excel 计算过去一年的收益。

L3-07：资产配置

讲授 + 实操 3 课时

内容：股票、债券、商品、现金的配置逻辑。60/40 组合。相关性分散。

通达信知识体系对应：第二篇 2.1~2.2（行业格局、四强对比）

练习：为一位"拥有 1000 万可投资资产的企业主"设计资产配置方案。

L3-08：综合项目 + 认证

项目 + 考核 3 课时

产出：一份《个人投资系统白皮书》——你的交易规则、风控规则、资产配置逻辑。

答辩：30 分钟 presentation + Q&A。评委包括导师和企业家校友。

L3 通关条件：白皮书 ≥ 75 分 + 答辩通过。

L4 · 量化与资产管理

18~22 岁 / 企业二代 · 24 课时 · 核心目标：掌握量化思维，理解现代资产配置，为接班做准备

5.1 L4 课程大纲

L4-01：量化交易平台实战

讲授 + 实操 3 课时

内容：安装配置 miniQMT，Python 基础速通（面向量化场景）。

通达信知识体系对应：第三篇 3.2~3.4（QMT vs PTrade、系统架构）

练习：用 xtquant 获取实时行情，打印到控制台。

L4-02：从通达信信号到自动交易

讲授 + 实操 3 课时

内容：通达信条件预警输出 → 文件监控 → Python 信号解析 → 自动下单。

通达信知识体系对应：第三篇 3.5~3.6（数据流、关键实现）

练习：跑通完整的"通达信→miniQMT"本地链路（模拟盘）。

L4-03：策略回测与评估

讲授 + 实操 3 课时

内容：回测框架搭建、过拟合识别、样本外测试、绩效评价指标（夏普、卡玛、胜率、盈亏比）。

练习：对一个简单均线策略做回测，计算绩效指标。

L4-04：量化风控体系

讲授 + 实操 3 课时

内容：三层风控防线、资金管理、熔断机制、黑天鹅应对。

通达信知识体系对应：第三篇 3.7（风控体系）

练习：为 L4-02 的策略编写完整的 Python 风控模块。

L4-05：行业竞争分析

讲授 + 项目 3 课时

内容：量化行业格局、机构 vs 个人、利基市场分析。

通达信知识体系对应：第三篇 3.9~3.10（竞争差异分析、利基市场）

讨论："如果我是管理 1 亿资金的基金经理，我的竞争策略是什么？"

L4-06：企业财务与资产配置

讲授 + 项目 3 课时

内容：企业资金管理、流动性规划、闲置资金投资策略、家族资产配置。

案例：某实业企业年利润 5000 万，如何规划"产业 + 投资"的双轮驱动？

练习：设计一份家族资产配置方案（含不动产、金融资产、实业、现金）。

L4-07：金融科技前沿

讲授 3 课时

内容：AI 在投资中的应用、金融大模型介绍、信创与国产替代趋势、监管政策影响。

通达信知识体系对应：第二篇 2.3~2.4（政策刚需、信创先发优势）

讨论：AI 会取代基金经理吗？什么是 AI 做不到的？

L4-08：毕业项目 + 认证

项目 + 考核 3 课时

产出：一份完整的《量化投资策略 + 资产配置方案》报告。

答辩：45 分钟 presentation，面向导师团 + 企业家评委。

L4 通关条件：答辩通过可获得通达信认证 Analyst 证书。

考核与认证体系

过程评估 + 终结考核，确保学习效果

6.1 评估方式

类型占比说明
课堂参与20%出勤、讨论、小组合作质量
平时作业30%每节课后通达信操作练习 + 简短反思
阶段项目30%各级别的产出（投资组合/分析报告/策略白皮书/量化方案）
期末答辩20%面向导师团的 presentation + Q&A

6.2 通达信操作技能考核标准

级别考核项目要求
L1基础操作开机登录、搜索股票、看 K 线、看 F10、加自选
L2分析工具快捷键操作、条件选股、板块分析、指标切换、数据导出
L3公式与交易编写简单选股公式、设置预警、Level-2 盘口解读
L4量化系统通达信→miniQMT 链路搭建、Python 策略编写、风控配置

6.3 认证体系

🥇 通达信认证 Analyst

L1~L4 全通关后获得。

  • 认证内容：通达信全功能使用 + 投资分析框架 + 量化基础
  • 证书形式：纸质证书 + 数字徽章
  • 附加权益：优先推荐至合作金融机构实习

📜 单级结业证明

每个级别通过后可获得。

  • 适合中途加入或按需学习的学员
  • 记录该级别掌握的技能清单
  • 可作为综合素质评价材料提交

学员成长路径

🌱

L1 少年金融启蒙

12~16岁，零基础。建立了正确的金钱观，知道股票、基金、债券的基本概念，会使用通达信看行情。

🌿

L2 投资分析入门

14~18岁。熟练使用通达信，能看懂 K 线和基本技术指标，会分析 F10 资料，能写简单的投资分析报告。

🌳

L3 实战交易训练

16~22岁。建立了自己的交易系统，理解风险管理，会写简单的通达信公式，具备初步的量化思维。

🏆

L4 量化与资产管理

18~22岁 / 企业二代。掌握量化交易基本技能，理解资产配置逻辑，能设计家族资产配置方案。获得认证 Analyst 资格。

🎯

    DeepSeek V4 情感分析
    对筛选出的品种，调 DeepSeek V4 Flash API 自动分析新闻情绪。 注意：你当前正在用的就是同一个模型——Claude Code CLI 背后就是 DeepSeek V4。
    # 运行时调 DeepSeek API（和 CLI 是同一个模型） from openai import OpenAI client = OpenAI( api_key=os.getenv("DEEPSEEK_API_KEY"), base_url="https://api.deepseek.com" ) prompt = f"""近一周 {品种} 的新闻： {news_text} 请判断：利好 ✅ / 利空 ❌ / 中性 ⏸ 一句话说明理由。""" resp = client.chat.completions.create( model="deepseek-chat", messages=[{"role":"user", "content": prompt}] )
  • DeepSeek V4 定价极低：每月 1 万次分析成本不到 ￥50
  • 开发时：终端里跟 Claude Code CLI 对话，它帮你写量化代码
  • 运行时：Python 代码直接调 DeepSeek API 做新闻分析
    • DeepSeek V4利好/利空￥20-80/月

课程导论：为什么这 8 周值得

三门必修课

作为企业接班人，你的"投资"不只是买卖股票——而是管理家族财富、配置企业资产、理解资本市场。这门压缩课程的核心不是教你怎么炒股，而是帮你建立一套可传承的资产管理框架。

四个模块的逻辑

模块解决的问题对企业的价值
一"我看不懂市场在发生什么"能独立判断市场环境，看懂分析师报告
二"我做决策总是受情绪影响"建立系统化决策框架，减少非理性失误
三"我想用程序辅助投资但不知从何入手"掌握量化思维，能用工具验证投资想法
四"家族资产和企业资金应该怎么管"能设计资产配置方案，与专业投顾有效沟通

一

市场认知与工具

2 课 · 看懂市场在发生什么，用通达信高效获取信息

第 1 课 市场格局与通达信高阶运用

讲授 + 实操 3 课时

本课目标

  • 理解中国资本市场格局——券商、公募、私募、游资各在做什么
  • 通达信从"会用"到"精通"——信息获取效率翻倍
  • 建立每日市场扫描的标准化流程

第一部分：中国资本市场格局（45min）

  • 市场参与者全景：散户（交易量占比 ~60%）、公募基金（~30 万亿规模）、私募量化（~1.5 万亿）、游资、外资
  • 信息在市场中如何传递？谁先知道？谁后知道？
  • 作为企业二代，你在市场中的定位——不是跟散户拼速度，是跟机构比认知

第二部分：通达信高阶操作（60min）

  • 多股同列、多周期同列、自定义页面布局
  • 板块监控与资金流向分析
  • 条件预警的进阶用法（多条件组合、文件输出）
  • 快捷键强化：日常扫描全市场只需要 10 秒

第三部分：每日市场扫描流程（45min）

开盘前（15 分钟）

  • 外盘表现（美股/A50/恒生）
  • 当日重要事件（经济数据/政策公告）
  • 自选股预警检查

盘中（3 次/日）

  • 大盘走势判断
  • 板块涨幅排名（寻找热点）
  • 持仓/关注品种扫一眼

收盘后（30 分钟）

  • 当日复盘（大盘/板块/个股）
  • 交易记录检查
  • 明日计划

课后任务

  • 在通达信中搭建自己的"工作台"（自选股分组 + 自定义版面）
  • 按上述流程执行 3 个交易日，记录心得体会

第 2 课 技术分析与公式编写

讲授 + 实操 3 课时

本课目标

  • 理解技术分析的核心哲学（不是预测，是应对）
  • 掌握通达信公式编写基础，能自定义指标
  • 避免技术分析中最常见的陷阱

第一部分：技术分析的本质（45min）

  • 三大假设：市场行为包容一切、价格沿趋势移动、历史会重演
  • 技术分析不是算命——它是一种风险管理工具
  • 企业二代的视角：技术分析在资产配置中的角色（择时 vs 择仓）

第二部分：通达信公式编写（90min）

  • 公式语言基础：变量、运算符、内置函数（MA, EMA, CROSS, HHV, LLV, REF）
  • 实战：编写一个自定义选股公式——"价量齐升+均线多头排列"
  • 实战：编写一个技术指标公式——自定义 MACD + 成交量确认
  • 公式调试与优化技巧

第三部分：常见陷阱（30min）

  • 过度拟合：把参数调到完美拟合过去，但未来完全失效
  • 幸存者偏差：只看成功的案例，忽略失败的
  • 未来函数：不小心用到未来数据，回测结果虚假
核心原则：技术指标是你决策的"参考系"，不是"圣杯"。同样的 K 线形态在牛市中成功率 70%，在熊市中可能只有 30%。先判断市场环境，再用指标——这个顺序不能反。

课后任务

  • 编写 1 个自己的选股公式，理由写清楚
  • 用通达信的回测功能测试过去 3 个月的表现
二

交易系统构建

2 课 · 管好情绪，管好风险，建立可复用的交易系统

第 3 课 交易心理学与行为金融

讲授 + 案例 3 课时

本课目标

  • 认识最常见的投资心理偏误，建立免疫系统
  • 理解为什么聪明人也会做蠢事
  • 企业二代的特殊心理挑战（家族压力、光环效应、风险偏好偏移）

第一部分：行为金融学核心（60min）

损失厌恶

亏 100 的痛苦 ≈ 赚 250 的快乐。这是人类天性，不是你的错。但知道它就能对抗它。

确认偏误

只找支持自己判断的证据，忽略反面信息。持有股票时就只看利好新闻。

锚定效应

被第一个看到的价格锚住。"这只股票之前是 100 块，现在 60 块，好便宜！"——可能它只值 30 块。

处置效应

过早卖出盈利的股票，死扛亏损的股票。结果：赚小钱，亏大钱。

第二部分：企业二代的特殊心理挑战（45min）

  • 光环效应："我爸做实业很成功，我炒股也应该赚"——实业和金融是两套能力
  • 家族压力：亏了会被评判，导致不敢做决策或过度冒险
  • 风险偏好偏移：用别人的钱（家族资金）和用自己的钱，决策质量不同

第三部分：构建决策防火墙（45min）

  • 写投资日记——每笔交易记录时写下"我的情绪状态"
  • 设检查点——在情绪激动（大涨/大跌）时不交易，等 24 小时
  • 请外脑——和有经验的投资者定期交流，暴露自己的盲区

课后任务

  • 开始写投资日记（每天 5 分钟），重点记录情绪状态
  • 找出自己过去最亏钱的一笔交易，分析当时是什么心理偏误导致的

第 4 课 风险管理与仓位控制

讲授 + 实操 3 课时

本课目标

  • 理解风险管理是投资的第一原则——不是收益最大化，是生存
  • 掌握仓位管理和止损止盈的具体方法
  • 设计自己的风控规则

第一部分：风险管理的底层逻辑（45min）

  • "不要亏大钱"比"赚大钱"重要 100 倍——亏损 50% 需要赚 100% 才能回本
  • 最大回撤是资产管理能力的核心指标——机构考核基金经理先看回撤再看收益
  • 企业二代的特殊风险：家族资金的安全边际要求更高

第二部分：仓位控制方法（60min）

方法规则示例适合场景
固定比例法单品种 ≤ 总资金 20%，总仓位 ≤ 70%初学者，简单易执行
凯利公式f = (p × b - q) / b，根据胜率和赔率计算有历史数据的策略
波动率调整高波动品种仓位减半，低波动品种仓位加倍多品种组合
最大回撤止损账户总回撤 -10% 时清仓休息所有场景的底线

第三部分：止损的艺术（45min）

  • 价格止损：跌破某价位就走（如 -5% 或跌破 20 日均线）
  • 时间止损：买了以后 N 天不涨就走（资金效率）
  • 事件止损：买入逻辑被证伪（如公司业绩变脸）
  • 熔断机制：当日亏损达到 X% 时停止所有交易

课后任务

  • 设计一套自己的交易风控规则（书面，不少于 5 条）
  • 在通达信中设置条件预警，用技术手段辅助执行
三

量化交易实战

2 课 · 从想法到策略，用程序验证投资逻辑

第 5 课 量化平台与自动化交易

讲授 + 实操 3 课时

本课目标

  • 理解量化交易的核心逻辑——规则化、系统化、自动化
  • 了解 QMT / miniQMT 平台的基本用法
  • 体验"通达信信号 → 自动下单"的完整链路

第一部分：量化交易是什么（30min）

  • 量化 ≠ 高频交易。量化 = 用规则替代直觉
  • 量化的三层价值：纪律性（不情绪化）、可验证（能回测）、可改进（迭代优化）
  • 国内外量化行业格局：幻方/九坤/明汯在做什么，个人量化怎么做
  • 企业二代的视角：量化思维不仅用于交易，也用于企业决策——用数据替代直觉

第二部分：QMT / miniQMT 实战（90min）

  • QMT 与 PTrade 的定位对比：本地运行 vs 云端运行，各有什么优劣
  • miniQMT 安装配置：极简模式启动，Python 环境准备
  • 核心函数演示：get_market_data() 获取行情、passorder() 下单
  • 现场演示：通达信条件预警 → 写入 JSON 文件 → Python 监控 → 自动下单

第三部分：完整链路演示（30min）

链路全景

通达信（指标公式→条件预警→信号文件）→ Python 监控（watchdog→解析→风控）→ miniQMT（xttrader.passorder→委托）→ 券商柜台 → 交易所

课后任务

  • 在自己的电脑上安装 miniQMT 并完成配置
  • 运行提供的示例 Python 脚本，获取实时行情
  • （选做）尝试将通达信的一个条件预警输出到文件

第 6 课 策略回测与绩效评估

讲授 + 实操 3 课时

本课目标

  • 学会用历史数据检验策略的有效性
  • 理解关键的绩效评估指标
  • 识别回测中最常见的"作弊"手段

第一部分：回测框架与方法（60min）

  • 回测的基本流程：定义策略 → 历史数据跑一遍 → 计算绩效 → 分析
  • 样本内 vs 样本外：留一部分数据不做优化，最后验证
  • 过拟合识别：如果策略参数稍微一变结果就差很多，说明过拟合了
  • 实战演示：用 Python 对均线策略做回测

第二部分：绩效指标解读（45min）

指标公式/含义说明
年化收益率总收益换算到年基本指标，但单独看没有意义
最大回撤从最高点到最低点的最大跌幅最重要的风控指标
夏普比率(收益-无风险利率)/波动率衡量每单位风险获得的超额收益，>1 算好
胜率盈利交易次数/总次数的比例高胜率不一定是好策略（很多小赚+一次大亏）
盈亏比平均盈利/平均亏损胜率 40% 但盈亏比 3:1 也能赚钱

第三部分：常见回测陷阱（30min）

  • 前视偏差：用到了未来才发生的数据（如财报发布后才知晓的数据提前使用了）
  • 忽略交易成本：没算手续费、滑点、冲击成本
  • 生存偏差：只测试还在的股票，退市的那些亏损被忽略了

课后任务

  • 用通达信或 Python 回测一个简单策略（均线/KDJ/RSI 任选），计算全部指标
  • 写一份简短的策略评估报告（含策略逻辑、回测结果、风险和局限性）
四

资产管理与接班

2 课 · 企业二代专属内容，从投资到资产管理

第 7 课 家族资产配置与企业资金管理

讲授 + 实战 3 课时

这是整门课程最核心的一课

前面的 6 课都在帮你建立"投资能力"，这一课把它落地到"资产管理"——作为企业接班人，你真正需要的是管理家族财富的能力，而不仅仅是买卖股票的能力。

第一部分：资产配置的底层逻辑（45min）

  • 资产配置决定了投资组合 90% 以上的收益波动（Brinson 研究）——选股和择时没那么重要
  • 核心资产类别特性对比：股票（高收益高波动）、债券（稳定收益低波动）、商品（对冲通胀）、现金（安全性）、不动产（抗通胀但流动性差）
  • 相关性分散——把不相关的资产放在一起，降低整体波动

第二部分：家族资产配置实战（60min）

案例背景：某制造业家族企业，年利润 5000 万，现有可投资金融资产 8000 万，不动产 3000 万，企业估值约 5 亿。企业主 55 岁，计划 5 年内逐步交班给孩子。

问题 1：流动性分层

  • 第一层：日常备用金（3~6 个月开支）
  • 第二层：中期配置（1~3 年，债券+分红股票）
  • 第三层：长期增长（3 年以上，权益类+私募股权）

问题 2：产业与投资的平衡

  • 实业现金流稳定时，金融资产可以更进取
  • 实业需要资金时，金融资产提供流动性
  • "产业 + 投资"双轮驱动如何落地

问题 3：交班过渡期配置

  • 逐步降低风险偏好（随着年龄增长）
  • 建立家族办公室或委托专业投顾
  • 子女参与投资决策的路径设计

第三部分：与专业投顾共事（30min）

  • 如何选择投顾机构——看业绩不如看理念一致性
  • 如何评估投顾表现——不是每年排名前 10%，是风险调整后收益 + 理念执行
  • 投顾 vs 家族办公室 vs 自己管——不同规模下的最优选择

第四部分：结业项目启动（15min）

项目任务：以自己家族的实际状况（或一个假设场景），设计一份完整的家族资产配置方案。第 8 课进行答辩。

项目要求：包含现状分析、配置比例、选择理由、风险评估、与投顾合作模式。不要求真实数据（可用脱敏数据），但逻辑必须自洽。

第 8 课 结业项目答辩

项目 + 考核 3 课时

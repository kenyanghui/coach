Mac 用通达信总览

通达信是 Windows 平台的软件，没有官方 Mac 版本（只有功能阉割的"金融终端 Mac 版"）。如果你想在 Mac 上获得完整的通达信体验，需要用一些"曲线救国"的方案。

核心问题：M 芯片（M1/M2/M3/M4）的 Mac 不能安装 Windows 双系统（Boot Camp 仅支持 Intel Mac）。 所以 M 芯片用户必须靠虚拟机或兼容层。

我该选哪个方案？

  • M 芯片 + 需要完整功能 + 愿意付费 → Parallels Desktop (最佳体验)
  • M 芯片 + 完整功能 + 不想花钱 → VMware Fusion (免费够用)
  • 只看盘、不交易、不装虚拟机 → 通达信 Mac 原生版 (最简单)
  • Intel Mac + 追求极致性能 → Boot Camp (原生 Windows)
  • 不想装 Windows、轻度使用 → CrossOver

方案一：Parallels Desktop 最推荐

▸ 为什么推荐

Parallels 是在 Mac 上运行 Windows 体验最好的虚拟机软件。它的 融合模式 (Coherence Mode) 能让 Windows 程序直接显示在 Mac 桌面上，就像原生应用一样。

▸ 安装步骤

  1. 下载 Parallels Desktop — 官网 parallels.com 下载（支持 14 天免费试用）
  2. 安装 Windows — 启动 Parallels，它会引导你自动下载并安装 Windows 11（自动处理，约 20~30 分钟）
  3. 下载通达信 — 在虚拟机里打开浏览器，去券商官网或 tdx.com.cn 下载 Windows 版通达信
  4. 安装通达信 — 正常安装，和在 Windows 电脑上一样
  5. 开启融合模式 — 点击 Parallels 菜单 「查看 → 进入融合」，通达信窗口就会脱离虚拟机界面直接出现在 Dock 和桌面上
融合模式的效果：通达信的窗口会出现在 Mac 桌面上，Dock 栏会有它的图标，你甚至可以用 Cmd+Tab 切换它。除了界面还是 Windows 风格，使用体验和原生 Mac 应用几乎没区别。

▸ 关键设置优化

  • 内存分配：给虚拟机分配 4GB~8GB 内存（Parallels 会自动推荐合适的值）
  • CPU 核心：分配 2~4 个核心（M 芯片性能足够）
  • 共享文件夹：设置 Mac 的「下载」或「桌面」文件夹共享给 Windows，方便导数据
  • 自动启动：可设置 Mac 开机时自动启动 Parallels 和通达信
注意：Parallels 是付费软件。订阅版约 ¥500/年（含免费升级），也有买断版但较贵。建议先试用 14 天确认满足需求再购买。

方案二：通达信 Mac 原生版

通达信官方有一个 Mac 版金融终端，可以直接在 macOS 上运行，不需要虚拟机。

▸ 优点

  • 不需要安装 Windows，不需要虚拟机，最省事
  • 原生 Mac 应用，功耗低、不发热
  • 免费使用

▸ 缺点

  • 功能不全：缺少很多 Windows 版的核心功能
  • 不支持公式编写、DLL 插件
  • 不支持自定义版面
  • 部分技术指标缺失
  • 更新频率低，版本落后
  • 部分券商不支持 Mac 版交易

▸ 下载与安装

  1. 打开浏览器访问 tdx.com.cn 或搜索"通达信 Mac 版"
  2. 下载 .dmg 安装包
  3. 双击打开，将通达信拖入「应用程序」文件夹
  4. 首次打开如果提示"无法验证开发者"，去 系统设置 → 隐私与安全性，点击"仍要打开"
适合场景：如果你只需要看看 K 线、分时图、自选股，不涉及复杂分析和交易，Mac 原生版完全够用。它就像一个"简化版通达信"。
券商定制 Mac 版：部分券商（如中信、华泰、招商证券等）会提供自己的 Mac 版交易终端，功能比官方通用 Mac 版更完整，建议优先用券商提供的版本。

方案三：VMware Fusion 免费

VMware Fusion 是 Parallels 的主要竞争对手。从 2024 年起，VMware Fusion 对个人用户免费提供。

▸ 安装步骤

  1. 下载 VMware Fusion — 访问 support.broadcom.com 注册个人账号下载 VMware Fusion Pro（免费）
  2. 获取 Windows — 你需要自己准备 Windows 11 ARM 版镜像（可从微软官网下载）
  3. 创建虚拟机 — 拖入 Windows 镜像文件，按向导完成安装
  4. 安装通达信 — 在虚拟机中正常下载安装

▸ 与 Parallels 的对比

总结：VMware Fusion 免费且性能足够，适合愿意花点时间配置的用户。如果你不想折腾，Parallels 更省心。

方案四：CrossOver

CrossOver 不是虚拟机，而是一个兼容层——它让 Windows 程序直接在 macOS 上运行，不需要安装 Windows 系统。

▸ 优点

  • 不需要 Windows 授权
  • 不需要完整的 Windows 系统，省磁盘空间（仅需 2~3GB）
  • 程序启动快，没有 Windows 启动过程

▸ 缺点

  • 兼容性不确定——不是所有 Windows 程序都能正常运行
  • 通达信的部分功能可能有问题（尤其是交易模块）
  • 性能不如虚拟机
  • 出现问题难以排查

▸ 安装步骤

  1. 在 codeweavers.com 购买并下载 CrossOver（约 ¥200，有 14 天试用）
  2. 打开 CrossOver，点击「安装 Windows 软件」
  3. 搜索通达信（或直接选择"安装未列出的应用程序"）
  4. 浏览选择通达信的安装包 setup.exe
  5. 按提示完成安装，然后从 CrossOver 的 Bottle 中启动通达信
注意：CrossOver 运行通达信的体验因版本而异。建议先用试用版测试你的券商版本是否正常工作，再决定是否购买。

方案五：Boot Camp（仅 Intel Mac）

如果你的 Mac 是 Intel 芯片（2020 年之前机型），可以使用苹果自带的启动转换助理安装 Windows 双系统。

▸ 安装步骤

  1. 打开「应用程序 → 实用工具 → 启动转换助理」
  2. 设置 Windows 分区大小（建议至少 80GB，通达信加数据可能占用不少空间）
  3. 插入 Windows 10/11 USB 安装盘（或选择 ISO 文件）
  4. 按提示重启并安装 Windows
  5. 安装完成后，在 Windows 中安装通达信
切换系统：开机长按 Option（⌥）键选择启动盘，或在系统设置中设置默认启动系统。
特点：Boot Camp 是完整的 Windows 系统，通达信 100% 功能可用，性能最佳。缺点是需要重启切换系统，无法和 macOS 应用同时使用。

其他方案

▸ 云电脑方案

如果你完全不想在本地装任何东西，也可以使用云电脑：

  • 华为云桌面 / 阿里云无影 — 远程连接一个 Windows 云桌面
  • 在云桌面里安装通达信，通过浏览器或客户端访问
  • 优点：不占用 Mac 资源，随时随地可用
  • 缺点：依赖网络，有延迟，月费约 ¥50~100

▸ Wine 手动方案

通过 Homebrew 安装 Wine，手动配置 Windows 兼容层。适合有技术基础的用户：

# 安装 Wine（通过 Homebrew）
brew install --cask wine-stable

# 运行通达信安装程序
wine tdx_installer.exe

但这个方案配置复杂，稳定性无法保证。

▸ 键盘映射

Mac 键盘和 Windows 键盘的按键对应关系：

WindowsMac（Parallels 默认）说明

Alt⌥ Option通达信中 Alt 快捷键可以直接用 Option
F5Fn+F5 或 ⌘+5Mac 默认 F 键是功能键，需按 Fn
Delete⌫ Backspace等价
Ctrl+P⌘+P通讯设置
优化建议：在 Parallels 设置中可自定义键盘映射。建议把 ⌘ 映射为 Ctrl，这样通达信的 Ctrl+D、Ctrl+T 等快捷键直接用 ⌘ 操作。

▸ 浏览器打开还是桌面端？

Mac 原生版和 Windows 虚拟机版共用同一个数据文件夹概念：

  • 自选股同步：在虚拟机中设置好自选股，导出配置文件，导入到 Mac 版。或使用通达信的云同步功能
  • 自定义公式：Windows 版编写的公式无法直接在 Mac 版使用（Mac 版不支持公式管理）

▸ 窗口管理

  • Parallels 融合模式：通达信窗口独立存在，可拖拽到任意桌面
  • Mac 分屏：可以把通达信放在一侧，Safari/Chrome 放在另一侧看资讯
  • Mission Control：给通达信分配独立桌面，用 ⌃ Ctrl+←/→ 快速切换

▸ 数据下载

无论哪种方案，在 Windows 虚拟机中操作时：

  1. 打开通达信，菜单 行情 → 盘后数据下载
  2. 勾选需要的市场（沪深）和数据类型（日线、一分钟线、分时图等）
  3. 点击"开始下载"

Mac 原生版同样支持盘后数据下载，位置相同。

▸ 数据共享（虚拟机 ↔ Mac）

  • Parallels：自动共享 Mac 的「桌面」「文档」「下载」文件夹
  • VMware：设置共享文件夹，路径通常在 /mnt/hgfs/
  • 导出数据：在通达信中按 ⌘ Cmd+E 导出 Excel，文件直接保存到 Mac 桌面

常见问题

▸ Q：M1/M2/M3 Mac 能用 Parallels 吗？

可以。Parallels 完全支持 M 芯片，会自动下载 Windows 11 ARM 版。ARM 版 Windows 可以运行 x86 程序（通过微软的 Prism 转译），通达信运行流畅。

▸ Q：M 芯片 Mac 用虚拟机，通达信会卡吗？

不会。M1/M2/M3/M4 的性能跑 Windows 虚拟机绰绰有余。通达信又不是大型 3D 游戏，日常看盘、切换 K 线、条件选股都非常流畅。

▸ Q：虚拟机里上网和交易安全吗？

安全。虚拟机中的 Windows 和 Mac 共享网络，和普通 Windows 电脑一样。建议在虚拟机中也安装杀毒软件（Windows Defender 即可），不要下载不明软件。

▸ Q：Mac 原生版能交易吗？

部分券商支持，部分不支持。需要去你的券商官网查看是否提供 Mac 版交易终端。如果不支持，交易功能会显示"未开通"或直接没有交易入口。

▸ Q：Parallels 融合模式切换不畅怎么办？

尝试在 Parallels 设置中关闭"显示 Windows 开始菜单"选项。如果融合模式下通达信窗口显示异常（如工具栏图标太小），可以在 Windows 中调整显示缩放为 125% 或 150%。

▸ Q：数据很大，快没空间了？

虚拟机磁盘可以压缩。在 Parallels 中：操作 → 压缩虚拟机。或者清理 Windows 中的临时文件（磁盘清理工具）。日线数据一般只保留最近 3~5 年即可。

▸ Q：Mac 原生版某个功能没有，我该怎么办？

Mac 原生版功能有限是正常的。如果你确实需要某个缺失的功能（如自定义公式、特定指标），就只能用虚拟机方案了。

▸ Q：Intel Mac 用 Boot Camp 还是 Parallels？

需要频繁使用通达信：Boot Camp（性能最好，但每次要重启）
需要同时用 Mac 和通达信：Parallels（不用重启，可融合模式同时使用）

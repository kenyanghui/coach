L1

少年金融启蒙

适合 12~16 岁 · 16 课时 · 8 次课 × 2 小时

建立兴趣 · 正确金钱观 · 通达信初体验

L1 课程概述

目标学员：12~16 岁，零基础。不需要任何金融或编程知识。

教学理念：兴趣引导为主，游戏化、体验式学习。70% 实操 + 30% 理论。

通达信知识体系对应：第一篇 1.1~1.5（通达信是什么、安装、界面、基本操作）。

L1 通关条件：完成投资组合报告 + 通达信基础操作测试（开机、搜股票、看K线、看F10）。

重要：L1 阶段绝对不鼓励真实投资。所有交易均为模拟。本金永久损失的风险意识从第一课开始强调。

L1-01 钱是什么？

讲授 2 课时

核心问题

  • 钱从哪里来？为什么一张纸可以买到东西？
  • 通货膨胀是什么——今天的 100 元和 10 年前的 100 元为什么不一样？
  • 银行在中间扮演什么角色？

课堂活动

  • 小组讨论："如果你有 100 万，你会怎么做？"
  • 观看通胀模拟动画，感受货币购买力的变化

通达信任务

  • 安装通达信（Windows/Mac 根据实际情况选择）
  • 认识界面——菜单栏、行情列表、K线图、状态栏
  • 搜索自己知道的公司名称（茅台、腾讯、阿里、苹果）

L1-02 储蓄 vs 投资

讲授 + 实操 2 课时

核心概念

  • 利息：把钱存银行，银行为什么要给你利息？
  • 复利："利滚利"的力量——爱因斯坦说过复利是世界第八大奇迹
  • 风险与收益：为什么存款利息低，股票可能赚更多也可能亏？

课堂活动

  • 复利计算器实验——每个月存 100 元，年化收益 5%，到 60 岁是多少？
  • 对比：存银行 vs 买基金 vs 买股票的 20 年差异

通达信任务

  • 打开基金板块，浏览货币基金、债券基金、股票基金的分类
  • 观察不同类型基金的历史收益差异

L1-03 什么是股票？

讲授 + 实操 2 课时

核心概念

  • 股票是公司的所有权凭证——买股票就是当股东
  • 股东的权利：分红、投票、公司赚钱你受益
  • 股价为什么会波动？

课堂活动

  • "开一家奶茶店"模拟游戏：全班出资开店 → 分股份 → 每月分红 → 有人想买你的股份

通达信任务

  • 搜索 10 家知名公司添加到自选股
  • 观察它们的股价和公司全称

L1-04 认识 K 线

讲授 + 实操 2 课时

核心概念

  • K 线的四个价格：开盘价、收盘价、最高价、最低价
  • 阳线（红色）= 收盘价 > 开盘价 = 上涨
  • 阴线（绿色/蓝色）= 收盘价 < 开盘价 = 下跌
  • 上影线、下影线、实体

课堂活动

  • 每组发一张股票的月度股价数据表，手动画 K 线图

通达信任务

  • 用 F5 在分时图和 K 线图之间切换
  • 用 ← → 方向键查看历史 K 线
  • 找一根很长的阳线和一根很长的阴线

L1-05 什么是公司？

讲授 + 实操 2 课时

核心概念

  • 上市公司做什么——营收、利润、员工人数
  • 最简单的公司画像：靠什么赚钱？赚钱吗？

课堂活动

  • 选一家熟悉的公司（苹果/麦当劳/伊利/华为），上网查它的业务

通达信任务

  • 按 F10 看公司资料
  • 找到"公司概况"和"主营业务"——它靠什么赚钱？

L1-06 市场是什么？

讲授 + 实操 2 课时

核心概念

  • 上证指数、深证成指是什么——不是一只股票，是一篮子股票的平均
  • "大盘涨了"是什么意思？
  • 牛市 vs 熊市

课堂活动

  • 模拟市场游戏：每组代表一个公司，全班交易形成股价

通达信任务

  • 输入 03+Enter 看上证指数
  • 输入 60+Enter 看沪深涨幅排名
  • 输入 94+Enter 看板块涨幅排名

L1-07 我的投资组合

实操 + 项目 2 课时

任务

  • 虚拟 10 万元资金
  • 选择 3~5 只股票构建投资组合
  • 每选一只股票必须写出"为什么选它"（至少 50 字）

通达信任务

  • 建立自选板块"我的组合"
  • 对每只股票设置价格预警

L1-08 投资报告会

项目 + 考核 2 课时

产出要求

  • 3 分钟演讲——"我的投资组合和理由"

考核标准

  • 逻辑清晰（40%）
  • 表达流畅（30%）
  • 有反思——如果跌了，你学到了什么（30%）
  • 注意：不考核收益率

L1 通关条件

  • 完成投资组合报告
  • 通达信基础操作测试（开机登录、搜股票、看 K 线、看 F10、加自选）
L2

投资分析入门

适合 14~18 岁 · 24 课时 · 8 次课 × 3 小时

掌握通达信分析工具 · 看懂技术与基本面

L2 课程概述

目标学员：14~18 岁，已完成 L1 或具备同等基础。

教学理念：技术面和基本面并重。不鼓励"炒短线"，每笔模拟交易持仓不少于 3 天。强调"逻辑比结果重要"。

通达信知识体系对应：第一篇 1.6~1.12 + 第二篇（快捷键、K线、指标、选股、行业分析）。

L2 通关条件：分析报告 ≥ 70 分 + 操作考试 ≥ 80 分。

L2-01 通达信高手之路

实操 3 课时

内容：快捷键强化训练、界面自定义、自选股管理、板块分类、条件预警设置。

知识体系对应：第一篇 1.4~1.6（界面布局、基本操作、快捷键大全）。

课堂活动：快捷键计时赛——老师报功能，学生用最快速度操作。每正确完成一项得 1 分。

课后练习：将快捷键表打印出来贴在电脑旁，每天练习 5 分钟。

L2-02 K 线语言

讲授 + 实操 3 课时

内容：经典 K 线组合形态——十字星（多空平衡）、启明星（底部反转）、黄昏星（顶部反转）、红三兵（持续上涨）、三只乌鸦（持续下跌）、锤子线、吊颈线。

知识体系对应：第一篇 1.7（K线图）。

课堂活动："K 线猜谜"——只看 K 线形态猜后续涨跌，然后揭晓真实走势，分析对错原因。

关键认知：K 线形态是概率，不是预言。同样的形态在牛市中成功率更高，在熊市中更低。

L2-03 技术指标入门

讲授 + 实操 3 课时

内容：MA（移动平均线：5日/10日/20日/60日）、MACD（趋势判断）、KDJ（超买超卖）、RSI（相对强弱）、VOL（成交量）。

知识体系对应：第一篇 1.8（技术指标）+ 第二篇 2.2（通达信指标评分9.8）。

练习：每人在通达信调出 MACD + KDJ，在历史 K 线上找"金叉"（买入信号）和"死叉"（卖出信号）的例子。

重点强调：指标不是预测未来的水晶球，而是描述过去的统计工具。没有任何指标能 100% 准确。建议 2~3 个指标互相验证。

L2-04 基本面分析入门

讲授 + 实操 3 课时

内容：F10 资料详解——财务数据（营收、净利润、增长率）、盈利能力（ROE、毛利率）、股东结构（前十大股东、机构持仓）、分红融资（历史分红记录）。

知识体系对应：第一篇 1.10（F10资料）。

课堂活动：对比两家同行业公司（如茅台 vs 五粮液，宁德时代 vs 比亚迪）。谁的财务更健康？谁的生意更赚钱？

L2-05 条件选股实战

实操 3 课时

内容：用通达信条件选股器筛选股票。多条件组合（技术面+基本面+量价）。选股结果保存为板块。

知识体系对应：第一篇 1.11（条件选股）。

课堂挑战："选出市盈率 < 20、净利润增长 > 20%、MACD 金叉的股票"——看谁选出的结果最合理，并解释为什么选这些。

L2-06 行业分析入门

讲授 + 实操 3 课时

内容：行业分类标准、板块轮动逻辑、热点识别方法。为什么有些板块涨有些跌？产业链分析入门。

知识体系对应：第二篇全（行业分析 + 四强对比 + 为什么行业选通达信）。

练习：用 94+Enter 看今日板块涨幅排名。选一个领涨板块，分析上涨原因，找出板块内涨幅最大的 3 只股票（龙头股）。

L2-07 模拟交易大赛

实操 + 项目 3 课时

规则：虚拟 50 万资金，模拟交易 4 周（本课为启动 + 复盘）。

要求：每笔交易必须写分析逻辑（技术面理由 + 基本面理由）。每日收盘后记录持仓。

复盘方法：每周一次小组复盘——你的交易有逻辑吗？哪里做对了？哪里错了？

L2-08 投资分析报告 + 认证考试

项目 + 考核 3 课时

产出：一份完整的个股分析报告（2000 字以上），包含：行业分析、公司基本面分析、技术面分析、投资结论与风险提示。

考核项目：

  • 通达信操作技能考试（30 分钟）
  • 分析报告答辩（每人 15 分钟）

L2 通关条件：分析报告 ≥ 70 分 + 操作考试 ≥ 80 分。

L3

实战交易训练

适合 16~22 岁 · 24 课时 · 8 次课 × 3 小时

深入理解市场运行规律 · 建立个人交易系统

L3 课程概述

目标学员：16~22 岁，已完成 L2 或具备同等基础。

教学理念：从"知道"到"做到"。强化纪律性，建立系统化的交易方法论。

通达信知识体系对应：第二篇全 + 第三篇 3.1~3.3（行业对比、量化平台概览）。

L3 通关条件：个人投资系统白皮书 ≥ 75 分 + 答辩通过。

L3-01 市场微观结构

讲授 3 课时

内容：交易所运作机制（集合竞价 vs 连续竞价）、订单类型（限价单/市价单/止损单）、盘口五档买卖挂单解读、Level-2 行情的十档和逐笔成交。通达信盘口实战观察。

L3-02 通达信公式编写入门

讲授 + 实操 3 课时

内容：通达信公式语言基础——变量定义、内置函数（EMA、CROSS、MA 等）、条件判断（IF/AND/OR）。编写第一个自定义指标和选股公式。

练习：编写"5 日均线上穿 20 日均线"的选股公式，跑出结果并验证。

L3-03 交易心理学

讲授 + 实操 3 课时

内容：行为金融学校心概念：恐惧与贪婪、锚定效应、确认偏误、损失厌恶（亏 100 的痛苦 > 赚 100 的快乐）、处置效应（卖盈持亏）。

活动："交易心理实验"——在模拟环境中体验大涨、大跌、踏空、卖飞的心理感受。回看自己在 L2 的模拟交易记录，分析哪些决策是情绪驱动的。

L3-04 风险管理

讲授 + 实操 3 课时

内容：仓位管理（单品种仓位上限 / 总仓位上限）、止损止盈设置方法、最大回撤概念、夏普比率（收益/风险）、凯利公式入门（最优下注比例）。

练习：设计一套自己的交易风控规则（如：单笔止损 -5%、总仓位不超过 70%、单品种不超过 20%）。

L3-05 期权期货入门

讲授 + 实操 3 课时

内容：期货基础（远期合约、标准化、保证金、逐日盯市、交割）、期权基础（看涨/看跌、权利金、行权价、到期日）。通达信中查看期货和期权行情。

安全提示：期权期货有杠杆，风险远大于股票。本课仅为认知教育，不鼓励实盘操作。

L3-06 量化思维入门

讲授 + 实操 3 课时

内容：什么是量化交易（用规则代替直觉）、回测的概念（用历史数据检验策略）、为什么需要规则化的交易系统（纪律性、可复制、可改进）。

活动：手工回测一个简单规则——"5 日均线上穿 20 日均线买入，下穿卖出"，用 Excel 计算过去一年的收益、胜率、最大回撤。

L3-07 资产配置

讲授 + 实操 3 课时

内容：核心资产类别（股票、债券、商品、现金、不动产）、经典的 60/40 组合、相关性分散的原理、不同经济周期下的资产表现。

练习：为一位"拥有 1000 万可投资资产的企业主"设计资产配置方案，写出配置比例和理由。

L3-08 综合项目 + 认证

项目 + 考核 3 课时

产出：一份《个人投资系统白皮书》——你的交易规则、选股逻辑、风控规则、资产配置逻辑、复盘方法论。

答辩：30 分钟 presentation + Q&A。评委包括导师和企业家校友。

L3 通关条件：白皮书 ≥ 75 分 + 答辩通过。

L4

量化与资产管理

适合 18~22 岁 / 企业二代 · 24 课时 · 8 次课 × 3 小时

掌握量化思维 · 理解现代资产配置 · 为接班做准备

L4 课程概述

目标学员：18~22 岁 / 企业二代，已完成 L3 或具备同等基础。

教学理念：从投资者视角升级到管理者视角。将投资能力与家族企业资产管理相结合。

通达信知识体系对应：第三篇全（量化交易、系统架构、风控、竞争分析）。

L4 通关条件：答辩通过，获通达信认证 Analyst 证书。

L4-01 量化交易平台实战

讲授 + 实操 3 课时

内容：量化交易生态概览（QMT vs PTrade vs 自建）、miniQMT 安装配置、Python 基础速通（面向量化场景：变量、列表、字典、循环、函数）。

练习：用 xtquant 获取实时行情数据，打印到控制台。

L4-02 从通达信信号到自动交易

讲授 + 实操 3 课时

内容：完整链路：通达信条件预警输出 → JSON Lines 信号文件 → Python 文件监控 → 信号解析 → 风控前置校验 → xttrader.passorder() 自动下单。

练习：跑通完整的"通达信→miniQMT"本地链路（模拟盘验证）。

L4-03 策略回测与评估

讲授 + 实操 3 课时

内容：回测框架搭建、过拟合识别（参数过多=拟合历史而非预测未来）、样本外测试（留一部分数据做验证）、绩效评价指标（夏普比率、卡玛比率、胜率、盈亏比、最大回撤）。

练习：对一个简单均线策略做回测，计算全部绩效指标并撰写回测报告。

L4-04 量化风控体系

讲授 + 实操 3 课时

内容：三层风控防线（前置校验→委托拦截→柜台风控）、资金管理参数（单笔上限/总仓位上限/频率上限）、熔断机制（日内亏损 X% 自动停盘）、黑天鹅应对（极端行情下的保护措施）。

练习：为 L4-02 的策略编写完整的 Python 风控模块，包含仓位计算、频率控制和熔断逻辑。

L4-05 行业竞争分析

讲授 + 项目 3 课时

内容：量化行业格局（T1 幻方/九坤 vs T2 中型私募 vs T3 个人）、差距量化（延迟/研发/数据/资金）、个人/小团队的真实优势（零研发成本/小资金灵活性/决策速度/合规自由度）、利基市场策略。

讨论："如果我是管理 1 亿资金的量化基金经理，我的竞争策略是什么？"

L4-06 企业财务与资产配置

讲授 + 项目 3 课时

内容：企业资金管理（经营性现金流 vs 投资性现金流）、流动性规划（备用金/短期投资/长期投资分层）、闲置资金投资策略、家族资产配置（不动产+金融资产+实业+现金）。

案例：某实业企业年利润 5000 万，如何规划"产业 + 投资"的双轮驱动？

练习：设计一份家族资产配置方案（含不动产、金融资产、实业、现金的比例和理由）。

L4-07 金融科技前沿

讲授 3 课时

内容：AI 在投资中的应用（自然语言处理读财报、图像识别看图表、机器学习做预测）、金融大模型介绍（同花顺问财、东方财富妙想）、信创与国产替代趋势（通达信在信创中的先发优势）、监管政策对量化交易的影响。

讨论："AI 会取代基金经理吗？什么是 AI 做不到的？"

L4-08 毕业项目 + 认证

项目 + 考核 3 课时

产出：一份完整的《量化投资策略 + 资产配置方案》报告。

要求：

  • 策略部分：策略逻辑、回测结果、风控规则、Python 代码
  • 配置部分：家族资产配置方案（以企业二代身份撰写）、配置逻辑和预期收益

答辩：45 分钟 presentation，面向导师团 + 企业家评委。

L4 通关条件：答辩通过，获得通达信认证 Analyst 证书。

道商-天问堂｜期权期货·廖老师量化策略核心笔记

来源： 廖老师量化期权期货交流（2026-03-15） 整理日期： 2026-05-12 归属： 道商三要素之「投资-天问堂」

一、道商视角：期权期货在三要素中的定位

道商总纲明确：投资是天问堂，是底气，不是主业。

"天问堂易经量化交易，是你修行精进的底气，让你能松弛、有光泽、不为五斗米折腰。" "量化模型的判断标准：它是为了让你心安定，而不是让你更忙。"

期权期货量化交易属于道商三要素之「投资-天问堂」，其核心价值在于： - 稳定现金流，支撑培训（利他）和教练（忠义堂）深耕 - 让心安定，不必为生存焦虑，有资本选择做对的事 - 模型跑通后，打坐时不用惦记投资收益，这才是对的

二、基础知识：股票 vs 期货 vs 期权

1. 股票

特性 说明
方向 只能买涨（做多）
杠杆 无杠杆，100块买100块的货
涨跌停 每天最多涨/跌10%
最大亏损 退市=归零，但不会欠钱
核心特征 用自己的钱赌一个方向，亏完为止

2. 期货

特性 说明
本质 借钱投资（融资融券概念），带杠杆
举例 一手棉花期货需10万，只投1万=借了9万
方向 可以做多（赌涨），也可以做空（赌跌）
风险 方向错误+连续涨跌停无法平仓=穿仓，需额外赔偿，可能倾家荡产
历史案例 中国银行原油宝事件，客户赔几千万
结算方式 每日结算（实时结账），账户没钱让补，不补就清仓

廖老师态度： 不做期货，因为风险不可控

3. 期权（核心重点）

特性 说明
本质 类似"买保险"
类比 汽车险花1万，万一出事故几十万不用自己出

买方（廖老师只做这个）：风险有限，收益无限 - 最多亏掉全部本金（权利金） - 收益没有天花板，几十上百倍都有可能 - 类似买房交定金：房价涨了要，跌了定金不要了

卖方（坚决不做）：类似保险公司/赌场庄家 - 胜率85%，稳定赚钱（这是陷阱） - 但遇到黑天鹅行情风险无限，可能倾家荡产 - 常见骗局：忽悠老板投几百万做卖方，每月给十几万，碰到黑天鹅就赔死

廖老师原话： "我不做卖方，我只做买方。风险有底线。"

四、人性问题：为什么普通人拿不住大牛股？

大牛股上涨特征

大牛股上涨过程中会不断洗盘，每次上涨后回调约一半：

40 → 100 → 跌回60
200 → 跌回110
375 → 跌回220
1100 → 跌回500
2000 → 跌回1100
最后到4750

人性的考验

1万变20万（20倍）→ 又跌回10万 到2000时已经50倍，更舍不得卖 继续跌回一半 → 痛苦割肉

赚10倍、20倍后归零，普通人根本受不了

廖老师自述：

"我去年真的有一个单赚了20多倍，最后都没要，还是变成零了。人性这个东西，解决的方法只有量化。"

窖藏酒理论（时间之窖）—— 核心纪律

"就像封酒藏在地窖，30天不准动，动了酒就不香了。"

策略规则： - 买入的必须是百倍龙头 - 30天内不准操作，不准揭盖子 - 其他标的不能碰，一年只操作5-6个 - 每年约3-5个百倍龙头机会，每两个月买一个

为什么必须量化（机器执行）？ - 机器没有赚钱/赔钱的概念 - 赚一个亿对它没有影响 - 量化才能克服人心，严格执行规则

五、全球主流交易方法

方法一：突破边界法（追涨）

特性 说明
胜率 只有10%
优点 能抓到百倍大牛股、大龙头
缺点 容易假突破爆仓
风险控制 轻仓+以小博大
代表人物 利弗莫尔（个人资产曾达美国国库1/10）

投1万亏得起，赚一次10倍够亏10次

方法二：突破回撤法

特性 说明
胜率 90%
方法 不追涨，等突破后回调洗盘再买入
优点 安全性高
缺点 如果价格不回调，会错过大牛股
代表人物 K姐（粉丝3万，胜率90%）

六、量化模型开发规则

买点规则

  • 红线蓝线交叉（蓝线从下到上，红线从上到下）
  • 价格跌到红线附近 = 买点出现
  • 红黄标注 = 庄家吃货

合约筛选条件

  • 时间周期：10天到40天之内
  • 持仓量 ≥ 1000手
  • 选择现价三档的虚值合约
  • 用日线或120分钟线选股

卖出规则

  • 默认： 赚一倍自动平掉一半仓位（保本），剩下一半赚5-10倍全部卖出
  • 备选： 赚一倍卖一半，剩下持有满30天，不管结果全部卖出
  • 亏损处理： 买入后无论亏多少，30天内不准卖出（避免突发利好错过十倍收益）

风控规则（核心纪律）

  • 单品种投入 ≤ 总资金的5%
  • 同一板块（有色/化工/农产品等）只选一个品种
  • 模型胜率7-9成，严格执行可稳定盈利

AI接入要求

模型需接入AI，自动搜索当前品种最新信息，判断近期是否有重大利好或利空，辅助判断方向。

应用范围

  • 当前国内期货品种85个，后续新增辣椒、马铃薯、咖啡、电力等，总数超100个
  • 模型还可用于商品期权、股指期权、股票期权
  • 未来对接美国股票期权（24小时全球交易）

一、核心理念

1.1 心光使命

教在科创，育在自信，成在思维
帮助更多孩子心中有光，学业圆满！
混沌学园创办人李善友深度解读："自信是结果，思维是根。心光的'一'不是课程，是自信。"

1.2 教育观

  • 自信是真爱，成绩只是顺带——自信的孩子才能从容面对不确定的未来
  • 学习力比学历更重要——学历是暂时的，学习力是陪伴孩子终生的
  • 最高级的学习方法是创造学习方法本身——每个孩子都能创造自己的学习方法
  • 教在科创，育在自信——科创是载体，自信是内核

1.3 人才观

心光致力于培养马斯克型未来科技人才——具备AI创造力、学习力、思维力和自信力的青少年。

AI时代淘汰人的，一定不是AI，而是会用AI的人。
成为掌握AI的人，而不是被AI掌握的人。

二、微笑模型（感问挖建四步法）

微笑模型是心光独创的核心方法论，以培养马斯克型人才为目标。最新版为四步法：感→问→挖→建。

> **李善友洞察**："不是所有的'问'都是有价值的。好的'问'来自对系统的深度感知。这恰恰是AI做不到的——AI可以回答问题，但AI不能'看见'问题。"

2.2 三步详解

第一步：问（Ask）——使命驱动，发现问题

核心：引导孩子找到内心真正关心的问题，培养对世界的深度感知

| 维度 | 说明 |

|------|------|

| 核心问题 | "为什么这件事对我重要？""我想改变什么？" |

| 思维方式 | 黄金圈思维（Why→How→What） |

| 对应四营 | 爱创营（AI创新力）——从真实问题出发 |

| 关键培养 | 好奇心、问题意识、使命感知 |

| 教学工具 | 黄金圈画布、英雄之旅模型、5W2H提问法 |

我问故我在——问题驱动世界，好的问题是创新的起点。

第二步：挖（Dig）——第一性原理，探究本质

核心：用哲科思维挖掘问题的本质，剥离表象，回归基本原理

| 维度 | 说明 |

|------|------|

| 核心问题 | "这件事的本质是什么？""最根本的原理是什么？" |

| 思维方式 | 第一性原理、批判性思维、模型思维 |

| 对应四营 | 爱智营（AI思维力）——用思想来思考思想 |

| 关键培养 | 深度思考力、逻辑推理、本质追问 |

| 教学工具 | 第一性原理画布、思维导图、苏格拉底追问法 |

用思想来思考思想——模型思维是底层认知的核心，认知力就是创新力。

第三步：建（Build）——工程逻辑，落地实现

核心：将想法落地为可运行的项目，以产品经理的视角交付成果

| 维度 | 说明 |

|------|------|

| 核心问题 | "如何从0到1实现它？""怎样做得更简单？" |

| 思维方式 | 工程思维、极简设计、迭代思维 |

| 对应四营 | 爱习营（AI学习力）+ 爱信营（AI自信力） |

| 关键培养 | 执行力、创造力、表达力、自信心 |

| 教学工具 | MVP画布、PDCA循环、路演展示、AI工具链 |

工程逻辑与极简设计——在AI辅助下快速构建原型、测试迭代、交付成果。

2.3 三步与自信的闭环

微笑模型的终极呈现是自信——当孩子经历了完整的"感→问→挖→建"过程，三力合一，自信自然生长。


问（使命驱动）→ 我知道为什么而做
挖（本质探究）→ 我知道怎么做
建（落地实现）→ 我做到了
                ↓
          自信！我做到了！
李善友视角："微笑模型如果只作为课程设计框架，太可惜了。它应该成为孩子的思维操作系统——面对任何问题，第一反应都是问本质、挖原理、建方案。这比任何AI技能都持久。"

六、教学三原则

6.1 闻思修——每节课的标准节奏

每一节课按照"闻→思→修"三段式设计：

| 环节 | 时间占比 | 内容 | 教师角色 |

|------|---------|------|---------|

| 闻（听） | 25% | 知识输入、案例讲解、视频观看 | 讲授者 |

| 思（想） | 25% | 讨论分享、提问互动、批判思考 | 引导者 |

| 修（做） | 50% | 动手实践、AI创作、演讲呈现 | 教练 |

6.2 项目制学习（PBL）

从真实问题出发，经历完整创新周期：


选题 → 调研 → 设计 → 构建 → 展示 → 迭代

6.3 教练式教学

教师角色从"讲授者"转变为"教练"：

  • 引导——不直接给答案，而是引导学生自己找到答案
  • 反馈——及时、具体、建设性的反馈
  • 赋能——帮助学生看见自己的成长和可能性

九、李善友视角：心光体系的深度洞察与战略建议

来源：混沌学园创办人李善友深度解读心光课程体系
核心理念升华：教在科创，育在自信，成在思维
核心洞察：微笑模型不只是一套课程设计框架，更是孩子的「思维操作系统」

9.1 心光的"一"——自信

心光有几条产品线——集训营（四营）、教练课、春秋季班、研学。但心光的"一"是什么？

不是课程，是自信。

"教在科创，育在自信"——这句话就是心光的"一"。如果心光的一切产品都围绕"自信"这个点击穿，格局完全不同。不是为了卖课而卖课，是为了让每个孩子"心中有光"。课程是载体，自信是本质。

不是内容，是结构。不是知识，是自信。

9.2 微笑模型即思维操作系统

微笑模型的"感问挖建"四步，本质上是在帮孩子建立元认知——对思考的思考。这和在混沌追求的目标一致：不是教人思考什么，是教人如何思考。

微笑模型如果只作为课程设计框架，太可惜了。它应该成为孩子的思维操作系统——面对任何问题，第一反应都是：

  • 问：真正的问题是什么？我想改变什么？
  • 挖：这件事的本质是什么？最根本的原理是什么？
  • 建：如何从0到1实现它？怎样做得更简单？

这比任何AI技能都持久。

9.3 感知力——"问"的根基

好问题来自对系统的深度感知。AI可以回答问题，但AI不能"看见"问题——这正是心光教育的起点。

AI的边界，就是心光的起点。

AI不知道"什么才是真正重要的问题"，没有直觉，无法共情，无法感知系统中最微妙的张力。心光的差异化优势，不在于教孩子用AI，而在于教孩子在AI时代保护和发展人的独特性——感知力、直觉、同理心、意义感。这些是"问"的根基，也是创新的真正源泉。

不是所有的"问"都是有价值的。好的"问"来自对系统的深度感知。

9.4 从"教"到"教练"——AI智能体的战略意义

混沌创商院最大的突破，是从"我说你听"变成"你做我看"。

心光已经有"闻思修"教学设计——方向是对的，但还不够深。真正的教练不是告诉学生答案，是让学生在过程中自己发现答案。

战略建议：把"感问挖建"四步法做成AI智能体，让学生在课后和AI教练对话，而不是等每周一次的课程。

AI智能体教练的递进路径：


课后  → AI教练辅助练习（感问挖建引导）
日常  → AI教练陪伴学习（习惯养成）
学期  → AI教练评估成长（能力画像）
全年  → AI教练规划路径（个性化学习方案）

这样就从"课时交付"变成了"能力陪跑"——这是完全不同的商业模式和竞争壁垒。

9.5 第二曲线：从高端营到教育基础设施

心光目前的第一曲线是集训营/教练课——服务付费意愿高的家庭。但中国教育的结构性机会不在高端市场，在公立学校的普及。

广州七中的40分钟公益课、40分钟方案·初中版——这就是第二曲线的种子。如果能用AI智能体把"微笑模型"产品化，让任何一个公立学校老师都能拿来就用——那不是一门课，那是一个教育基础设施。

这正是混沌从"卖课"到"卖方案"的非连续性跨越——心光也需要思考：从"卖课"到"卖系统"的跨越点在哪儿？

9.6 心光的终极差异化：AI时代人的独特性

心光的差异化不在教孩子用AI工具——这个市场很快会变成红海。心光的真正差异化在：

教孩子在AI时代保持和发挥人的独特优势——感知、直觉、同理心、意义感、创造力。

这些是"感问挖建"的前提，也是AI无法替代的核心能力。心光培养的不是AI使用者，而是AI时代的创造者——而创造者的本质，是做AI做不到的事情。

9.7 最终寄语

混沌创商院做了7000+企业案例，创益院做了上百家公益组织。但说到底，教育这件事，不管是教创业者还是教孩子，"一"都是一样的：

让人成为更好的自己。

心光在做的事——用AI科创帮孩子心中有光——和混沌的追求本质上是一回事。只是触达的对象不同——你们触达的是孩子，我触达的是创业者。但最终，我们都在帮人完成一次认知升级，一次自信的觉醒。

教在科创，育在自信，成在思维。
帮助更多孩子心中有光，学业圆满！

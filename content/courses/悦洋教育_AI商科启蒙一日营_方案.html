悦洋教育 · AI商科启蒙一日营 方案

版本：v2.0（原1天AI创客启蒙课升级）
产品定位：入门级获客产品 — 低门槛体验AI商科，建立信任，为商科四营引流
价值主张：一天时间，让孩子看见商业世界的AI力量
定价：¥980/人（早鸟/拼团价¥680）

一、产品概况

1.1 产品定义

1天AI商科启蒙一日营是悦洋教育产品体系中的入门级产品，面向从未接触过商科或刚接触AI的青少年，通过6课时的沉浸式体验，让学员在一天之内完成从"商业是什么"到"用AI做商业创意"的完整认知闭环。

核心使命：不是教多少知识，而是点燃商业兴趣、建立商业思维、展示AI商科的可能性。

1.2 核心变化（v1→v2）

| 维度 | v1（AI创客启蒙） | v2（AI商科启蒙） |

|:----|----------------|----------------|

| 开场问题 | "AI能做什么？" | "商业是什么？AI怎么帮人赚钱？" |

| 核心活动 | AI绘画/音乐创作 | AI商业创意工作坊+商业模型初体验 |

| 项目产出 | 绘画/音乐作品 | 商业创意方案+AI工具应用 |

| 结束引导 | "用AI让世界更好" | "用AI创造商业价值" |

二、课程大纲（6课时/1天）

上午（3课时）：商业认知唤醒

第1课：商业是什么——与商业的第一次对话（1课时）

| 环节 | 内容 | 时长 |

|:----|------|:----:|

| 闻 | 什么是商业？商业如何创造价值？（展示有趣商业案例） | 15min |

| 思 | 讨论："生活中哪些商业你每天在用？" | 15min |

| 修 | 用AI分析一家你喜欢的品牌，了解它的商业模式 | 30min |

| 产出 | 每个学员用AI完成一个品牌分析卡片 | |

第2课：AI商业工具初体验（1课时）

| 环节 | 内容 | 时长 |

|:----|------|:----:|

| 闻 | AI如何帮企业赚钱？案例展示（AI营销/AI客服/AI数据分析） | 15min |

| 思 | 头脑风暴："这个生意能用AI做得更好吗？" | 15min |

| 修 | 用AI工具开一家"虚拟小店"——设计产品+定价+营销文案 | 30min |

| 产出 | 每人一个AI辅助的商业创意方案 | |

第3课：商业创意工坊（1课时）

| 环节 | 内容 | 时长 |

|:----|------|:----:|

| 闻 | 好商业创意的三个标准 | 15min |

| 修 | 小组协作，用AI工具完成一个商业创意方案 | 45min |

| 产出 | 小组商业创意海报 | |

下午（3课时）：项目体验

第4课：用AI完成一个商业项目（1.5课时）

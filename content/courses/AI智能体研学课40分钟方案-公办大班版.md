# AI智能体研学课 · 40分钟教学方案

**适用场景**：公办高中大班授课（40-50人）
**课程定位**：AI协作式研学体验课 / 公益课
**适用年级**：高一及以上
**设计迭代**：基于2026年5月8日广州七中真实课堂成果

---

## 一、设计理念

### 核心主张

> 不是替代老师，而是让老师插上AI的翅膀。

### 三条假设（已被验证）

| # | 假设 | 七中课堂验证结果 |
|---|------|-----------------|
| 1 | 高一学生不需先学AI操作，可以直接用AI创造 | ✅ 40分钟产出7个完整网站 |
| 2 | 给宏大框架比给具体指令更能激发学生 | ✅ 学生自发产出"称霸地球""制霸宇宙"等创意 |
| 3 | 建站工具是产出效率的关键杠杆 | ✅ MiniMax space 零门槛建站，学生5分钟上手 |

### 老师角色转变

```
传统课堂：老师讲 → 学生听 → 课后练
这堂课：  老师设框架 → 学生跟AI共创 → 课堂直接出成果
```

老师在这40分钟里不是"教AI"，而是：
- **架构师**：设计课题框架和任务卡
- **引导者**：巡场一句话激发学生
- **主持人**：把控节奏和展示环节

## 二、课前准备清单

### 2.1 技术准备（提前1天完成）

| 项目 | 说明 | 备注 |
|------|------|------|
| **飞书群 × 4个频道** | 建立班级飞书群，创建4个子频道 | 每个频道一个组 |
| **AI智能体配置** | 每个频道接入1个AI智能体 | 推荐使用能联网搜索+长上下文模型 |
| **建站模板 × 4** | 在 MiniMax space（或同类工具）创建4个空白网站模板 | 不同配色风格 |
| **任务卡 × 4** | 打印4张任务卡（见第六部分模板） | A4纸单面打印 |
| **计时器** | 准备40分钟倒计时大屏 | 用于控制节奏 |

### 2.2 分组策略（大班适用）

40-50人的班级分为 **4个大组**，每组10-12人：

```
组内角色分工（学生自选）：
┌─────────────────────────────────────┐
│  👤 提问官（1-2人）                  │
│     负责向AI智能体提问、追问          │
├─────────────────────────────────────┤
│  👤 整理官（2-3人）                  │
│     筛选AI回答，提炼要点              │
├─────────────────────────────────────┤
│  👤 视觉官（1-2人）                  │
│     负责网站排版、配色、配图           │
├─────────────────────────────────────┤
│  👤 内容官（3-4人）                  │
│     围绕课题组织内容框架              │
├─────────────────────────────────────┤
│  👤 发言人（1人）                    │
│     最后1分钟展示成果                 │
└─────────────────────────────────────┘
```

> **大班适配说明**：10-12人一组确保全员参与。角色由学生自选，不强求均衡，实践中学生会自然分工。

### 2.3 设备要求

| 设备 | 数量 | 用途 |
|------|------|------|
| 可上网电脑/平板/手机 | 每组至少2-3台 | 访问AI智能体 + 建站工具 |
| 教室大屏/投影 | 1台 | 计时器 + 展示环节 |
| 白板/黑板 | 1块 | 板书关键时间节点 |

## 三、40分钟课堂流程

```
时间轴：
0'─────────5'─────────10'──────────────────35'─────────40'
 │           │           │                    │           │
开场破冰    分组发卡    自由共创             成果展示    金句收尾
(3min)      (5min)     (25min)              (5min)      (2min)
```

### ⏱ 0:00-0:03 开场破冰（3分钟）

**老师话术参考**：

> "同学们，今天这40分钟，我们不做题、不听课——我们用AI做点有意思的事。
>
> 40分钟很短，打一局游戏就没了。
> 40分钟也可以很长，长到能从课堂走到宇宙。
>
> 你们分成4个组，每组有一个AI智能体搭档。你们负责想，AI负责帮你们做出来。
> 40分钟后，每组要拿出一个可以打开看的网站。
>
> 开始。"

**要点**：
- 不要演示AI怎么用——学生天然会用
- 不要讲AI原理——时间留给创造
- 制造紧迫感：40分钟倒计时开始

### ⏱ 0:03-0:08 分组 + 发任务卡（5分钟）

1. 快速分4组（如已提前分好，直接就位）
2. 每组发一张任务卡
3. 每组建站链接发到对应飞书频道
4. 学生自选角色（30秒内完成）

### ⏱ 0:08-0:33 自由共创（25分钟）

**学生自主流程**：

| 时间段 | 做什么 | 说明 |
|--------|--------|------|
| 0:08-0:13 | 组内讨论定方向 | 5分钟讨论，确定网站主题和内容框架 |
| 0:13-0:26 | 跟AI智能体对话 | 13分钟反复提问、追问、迭代内容 |
| 0:26-0:31 | 把内容做进网站 | 5分钟把AI产出的内容填入建站模板 |
| 0:31-0:33 | 最后修整 | 2分钟调整排版，准备展示 |

**老师巡场引导话术**（每30秒穿过一个组）：

| 场景 | 引导话术 |
|------|---------|
| 组内沉默 | "你们可以先问AI一个问题，看它怎么回答。" |
| AI第一版不满意 | "继续追问，让它换个角度重新说。" |
| 方向太散 | "选一个最亮眼的点，做深做透。" |
| 时间过半 | "还有12分钟，建议开始把内容搬进网站了。" |
| 快要完成 | "加一个让校长惊艳的数据或图表。" |

### ⏱ 0:33-0:38 成果快闪展示（5分钟）

每组 **1分钟** 展示：

```
展示模板：
┌─────────────────────────────────────┐
│  ① 报网址："我们的网站是xxx"        │
│  ② 秀亮点：翻到最得意的一页         │
│  ③ 一句话心得："我们学到了..."      │
└─────────────────────────────────────┘
```

老师逐组30秒点评：只说一个亮点。

### ⏱ 0:38-0:40 金句收尾（2分钟）

> "今天你们用40分钟证明了四件事——
>
> 第一，AI不是答案机器，它是你的思考伙伴；
> 第二，最厉害的AI提示词，是你对这个世界的真诚好奇；
> 第三，你们每个人都比想象中更有创造力；
> 第四，AI时代最稀缺的能力，是**做一个有温度的超级个体**。
>
> 网站链接会留在群里，你们可以继续迭代。
> 下课。"

## 四、4组课题设计

以下四个课题覆盖 **10个以上学科**，可直接用于课堂。

### 组一：战略组 —— "给七中设计AI时代战略"

| 项目 | 内容 |
|------|------|
| **课题名称** | 七中AI时代发展蓝图 |
| **任务描述** | 假设你是七中的战略顾问，请用AI设计一份七中在AI时代的发展战略 |
| **涉及学科** | 政治、历史、地理、语文 |
| **产出方向** | 战略规划网站，含定位、路径、里程碑 |
| **可用AI提示词示例** | "假设你是战略顾问，为广州七中设计一份AI时代的三年发展计划，包含：学校定位、师资建设、课程改革、品牌传播四个维度。" |
| **七中真实产出参考** | 《七中称霸地球战略》《七中AI制霸宇宙精华集》 |

### 组二：学科组 —— "为某学科制作AI教学指南"

| 项目 | 内容 |
|------|------|
| **课题名称** | XX学科AI教学指南 |
| **任务描述** | 选择一门学科，用AI设计一份教师AI应用指南，包含备课、授课、作业场景 |
| **涉及学科** | 全部10个学科（语文/数学/英语/物理/化学/历史/政治/地理/生物/信息） |
| **产出方向** | 学科指南网站，含工具推荐+Prompt模板 |
| **可用AI提示词示例** | "请为高一生物学设计一份AI教学指南，包含：细胞生物学、遗传与进化、稳态与调节、生态系统四个模块，每个模块给出备课AI工具和课堂互动方案。" |
| **七中真实产出参考** | 《七中各科AI应用方案》《高一生物AI教学完全指南》 |

### 组三：提分组 —— "用AI设计提分方案"

| 项目 | 内容 |
|------|------|
| **课题名称** | 用AI冲刺809分 |
| **任务描述** | 假设你的目标是年级前10名，请用AI分析差距、制定各科提分策略 |
| **涉及学科** | 数学、英语、语文、物理/化学等（全学科数据思维） |
| **产出方向** | 提分方案网站，含数据分析、分科策略、时间轴 |
| **可用AI提示词示例** | "假设我是高一学生，目前各科成绩在年级中游，目标是一年后进入年级前10。请帮我做：①各科差距分析 ②每科提分关键动作 ③每周时间分配方案 ④AI辅助学习工具推荐。" |
| **七中真实产出参考** | 《分数线提升至809分方案》 |

### 组四：创意组 —— "用AI做交互式学习工具"

| 项目 | 内容 |
|------|------|
| **课题名称** | XX知识点交互学习工具 |
| **任务描述** | 选择一个你学过的知识点，用AI把它变成一个有趣的交互式学习网页 |
| **涉及学科** | 数学、物理、化学、英语等（适合做交互的学科） |
| **产出方向** | 交互式学习网站，含知识点讲解+练习题+AI出题 |

## 五、大班授课关键要点

### 5.1 跟小班课的核心区别

| 维度 | 小班（15-20人） | 大班（40-50人） |
|------|----------------|----------------|
| 分组 | 4-5组，每组4人 | 4组，每组10-12人 |
| 角色分工 | 每人承担一个角色 | 更强调组内自发分工 |
| 老师巡场 | 可以逐个指导 | 每30秒一个组，一句话引导 |
| 展示环节 | 每组2-3分钟 | 每组严格1分钟 |
| 纪律管理 | 较容易 | 需靠节奏和倒计时拉动 |

### 5.2 大班授课风险预案

| 风险 | 概率 | 应对措施 |
|------|------|---------|
| 学生不会用AI工具 | 低 | 任务卡上印好提示词模板，直接复制提问 |
| 网站登录受阻 | 中 | 提前请学校信息老师协助开通网络权限 |
| 组内有人无事可做 | 中 | 角色分工卡明确每个人的责任 |
| 成果过于简单 | 低 | 巡场时引导追问："能不能让它再深一层？" |
| 时间不够用 | 中 | 每个环节可弹性压缩2-3分钟 |

### 5.3 简化版（设备不足时）

若遇到以下限制，可按对应方案简化：

**情况一：全校只有一个教师账号可用**
- 老师在大屏上操作AI，学生口述指令
- 各组派代表轮流提问
- 网站由老师统一操作，学生提供内容

**情况二：学生没有个人设备**
- 4组轮流使用2-3台设备
- 组内分工：提问组轮流操作，其余同学用纸笔整理内容
- 建站环节由每组派1名代表操作

**情况三：教室无网络/网络差**
- 课前在飞书群预置好AI智能体的对话种子
- 学生分组讨论确定内容方向
- AI产出环节改为老师统一在大屏操作演示
- 网站模板提前填好内容框架，现场只做调整

## 七、从七中课堂验证的迭代方向

### 已验证可复用的经验

1. **建站工具选型正确**：MiniMax space 零门槛，学生5分钟上手
2. **角色分工有效**：即使10-12人大组，自选角色后没有边缘人
3. **宏大叙事激发创造力**："称霸地球""制霸宇宙"等选题说明框架给了学生野心
4. **AI出题模式**：学生自发发现的用法，下次可直接写入任务卡

### 下次可尝试的改进

| 改进方向 | 说明 |
|----------|------|
| 增设"最佳追问奖" | 鼓励学生多轮迭代，而不是接受AI第一版答案 |
| 引入组间互评 | 每组分一个"评委"到其他组打分，增加参与感 |
| 预留"人类-only"环节 | 2分钟完全不用AI做头脑风暴，对比协作前后的差异 |
| 课后延伸挑战 | "用AI做一个家族故事网站"——连接家庭教育 |

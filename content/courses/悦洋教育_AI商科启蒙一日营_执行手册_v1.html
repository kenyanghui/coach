AI商科启蒙一日营 · 执行手册 v1.0

核心理念：6小时让孩子体验"用AI创造商业价值"——不是学AI工具，是体验一次当CEO。
定位：低门槛引流产品 → 筛选意向客户 → 转化到商科四营（预期转化率15-25%）

课程总览

| 项目 | 内容 |

|------|------|

| 时长 | 6小时（1天） |

| 设备 | 仅需手机 |

| 适合年龄 | 8-16岁（分组教学） |

| 每班人数 | 12-20人（4组，每组3-5人） |

| 师资配置 | 1主讲教练 + 2助教 |

| 核心工具 | 豆包 App（AI对话+生图）+ 灵光 App（作品集） |

| 定价 | ¥980/人（标准价），¥680/人（早鸟/拼团） |

| 核心成果 | AI商业海报 + AI商业原型 + 灵光作品集链接 + mini路演 |

分组逻辑

每组就是一个"创业公司"：

| 角色 | 人数 | 职责 | 选人标准 |

|------|:----:|------|---------|

| CEO（组长） | 1人 | 组织讨论+上台路演 | 表达能力强的孩子 |

| 产品经理 | 1人 | 负责产品创意+海报 | 有想象力的孩子 |

| AI工程师 | 1人 | 负责AI工具操作 | 动手能力强的孩子 |

| 商业分析师 | 1-2人 | 负责市场分析+成本核算 | 细心、数学好的孩子 |

教练注意：角色不是固定的。如果某个孩子对当前角色不适应，随时可以轮换。目标是让每个孩子找到自己的位置。

上午：发现商机，设计产品（3小时）

09:00-09:30 │ 报到破冰 · "开公司了！"

氛围目标： 让孩子从"上课"切换到"创业"模式。

  • 签到拍照（助教拍入营照）
  • 破冰游戏："一分钟CEO"
  • 每人用3句话介绍自己：名字 + 年龄 + "如果我是CEO，我要做_______"
  • 助教记录亮点（这些可能是下午奖项的依据）
  • 组建公司（分组）
  • 每组4-5人，选CEO
  • 起公司名 + 选角色
  • 公司名写在名牌上
  • 定规则：
  • "音乐响起回座位"——听音乐指令
  • "同学上台鼓掌"——相互尊重
  • "没有错答案"——大胆说

教练话术引入：

"今天，你们不是来上课的。你们是来开公司的。
每个组就是一家创业公司，你们要在6小时内：
发现一个商机 → 设计一个产品 → 做出一个原型 → 上台路演。
准备好了吗？"

09:30-10:30 │ 发现商机 · AI对话找灵感

目标： 每个孩子找到一个自己关心的商业问题。

环节1：商机启发（10min）

教练用大屏展示几个真实青少年创业案例：

| 案例 | 创始人年龄 | 解决的问题 |

|------|:---------:|-----------|

| 环保吸管 Strawby | 13岁 | 塑料吸管污染 |

| 二手校服平台 | 15岁 | 校服浪费 |

| 宠物自动喂食器 | 14岁 | 上班族没空喂猫 |

| AI学习助手 | 16岁 | 同学问同样的问题 |

教练引导： "这些问题可能就在你身边。你每天上学、放学、在家，有没有遇到过"太麻烦了"、"这个可以更好"的时刻？那就是商机。"

环节2：AI对谈找方向（15min）

打开豆包 App → 语音输入：


我今年 X 岁，我观察到________（描述一个你发现的问题/不方便的地方），
请采访我，帮我找到把这个问题变成商业机会的方向。
助教巡场： 帮助打字/语音输入困难的孩子。确保每个人至少完成1次对话。

环节3：小组分享（15min）

每人用1分钟讲自己的发现：

  • "我发现的问题是……"
  • "我想做一个……来解决它"

产出： 每个孩子确定一个商业方向。助教记录。

10:30-11:30 │ 设计产品 · AI生图做海报

目标： 把想法变成看得见的产品概念。

步骤1：确定产品概念（10min）

教练引导填表（用手机笔记或纸笔）：


我的产品名称：________
解决什么问题：________
目标用户是谁：________
我的Slogan：________

步骤2：用豆包生成产品海报（25min）

豆包 → AI生图 → 输入：


我是 X 岁的创业者____，
我的产品叫____，解决____问题，
目标用户是____，Slogan是____。
请帮我生成一张产品宣传海报，
风格现代、色彩鲜明、适合青少年。

→ 海报保存到手机相册。助教确保每人至少生成1张。

示范Prompt（教练可投屏展示）：

"我是13岁的创业者小明，我的产品叫'书桌整理助手'，解决学生书桌太乱找不到东西的问题，目标用户是中小学生，Slogan是'3秒钟找到你要的'。请帮我生成一张产品宣传海报，风格现代、色彩鲜明、适合青少年。"

步骤3：准备展示（10min）

每人准备30秒介绍自己的产品。可以写几个关键词当提示。

11:30-12:00 │ 团队互评 + 教练点评

互评规则： "说一个喜欢的 + 给一个建议"

每组1人上台展示（CEO或自愿者）：

  • 我们公司叫____
  • 我/我们做的产品是____
  • 它解决____问题
  • 下面是我们的海报

教练点评要点（只说好的，不否定）：

  • "这个产品很有创意，我没想到还能这样解决"
  • "你的目标用户想得很清楚"
  • "这个Slogan很有意思"

姓名 / 公司角色 / 产品方向 / 表达力 / 主动性 / 合作度 / 亮点备注

🍱 12:00-13:30 午休

  • 助教整理上午观察记录
  • 准备下午的课程材料
  • 检查灵光 App 是否正常

下午：商业原型与路演（3小时）

13:30-14:30 │ 商业原型（灵光）

目标： 用AI工具做出一个可以展示的商业原型。

什么是商业原型？

不是真的写代码做产品，而是模拟产品的核心功能——比如一段对话、一个简单的互动页面、一个产品介绍H5。

第一步：浏览灵感（5min）

灵光 → 创作广场 → 浏览精选作品。

教练大屏演示不同类型的互动作品。

教练话术：

"下午你要做的，是让你的产品'活起来'。不用写代码，用AI帮你做。哪怕只是一个简单的介绍页面+互动问答，都能让别人对你的产品有感觉。"

第二步：AI创作商业原型（25min）

灵光 → 创作 → AI创作 → 语音描述：

方案A — 产品介绍互动页（推荐）：


帮我做一个产品介绍页面，
我的产品叫____，解决____问题，
目标用户是____，Slogan是____。
帮我设计一个有互动功能的页面，
可以展示产品海报，让用户了解产品价值。

方案B — 商业问答游戏：


帮我做一个关于____的问答互动，
用户回答5个问题来了解这个产品，
风格____（色彩/风格），适合青少年。

第三步：分享+互玩（15min）

发布后生成链接 → 发到小组群 → 组内互玩：

  • "说一个最喜欢的点"
  • "给一个改进建议"

助教注意： 如果有孩子进度落后，助教1对1辅导。保存至少每人1个作品。

14:30-15:30 │ 商业计划 + 路演准备

目标： 整理全天成果，准备2分钟路演。

第一步：整理成果（10min）

每人准备：

  • 产品海报（手机相册）
  • 商业原型（灵光链接）
  • Slogan
  • 自我介绍的3个要点

第二步：搭建作品集（10min）

灵光 → 我的 → 我的作品集 → 新建 → "我的商业项目" → 添加海报图片 + 关联原型 → 保存 → 生成分享链接

教练检查点： 确保每个孩子都完成了作品集链接。这是全天最重要的交付物——家长现场扫码就能转发家族群。

第三步：路演准备 + 彩排（15min）

2分钟路演模板：


"大家好，我是____，是____公司的CEO。
我发现____问题，所以我想做____产品。
这是我的产品海报（展示），
这是我的商业原型（展示），
我的Slogan是____。
我们想____（未来的计划）。
谢谢大家！"

内向的孩子适配版（3句话即可）：


"我是____，我的产品是____，因为____。
谢谢！"

→ 助教在旁边展示海报和原型。

教练巡场要点：

  • 帮每个孩子压缩到核心信息
  • 练一遍（对着助教或组员）
  • 确认每个孩子都知道上台后说什么

奖项设计（教练提前标注每个孩子的奖项）：

| 奖项 | 适合什么样的孩子 |

|------|----------------|

| 🏆 最佳商业创意奖 | 想法最独特、最有创意的 |

| 🏆 最佳表达奖 | 口齿清晰、自信大方的 |

| 🏆 最佳AI应用奖 | 最会用AI工具的 |

| 🏆 最佳团队奖 | 最会合作的 |

| 🏆 社会关怀奖 | 关注社会问题的 |

| 🏆 未来CEO奖 | 领导力突出的 |

| 🏆 最佳进步奖 | 从害羞到敢说的 |

| 🏆 创意之星奖 | 脑洞最大的 |

原则： 每个孩子至少拿一个奖。奖项不是为了竞争，是为了让每个孩子都感受到"我被看见了"。

15:30-16:00 │ 路演展示 + 颁奖 + 家长参与

氛围布置：

  • 播放入场音乐（激昂的创业/科技类音乐）
  • "AI商科启蒙一日营 · 创造者发布会"（投屏）
  • 家长入座观众席

路演流程：

  • 每组依次上台（每组3-5分钟）
  • CEO做2分钟路演
  • 助教展示作品集链接（投屏）
  • 教练点评（只说亮点）
  • 下一组

颁奖环节：

  • 逐一念名字上台
  • 念奖项+一句具体肯定
  • 例："王小明获得最佳商业创意奖——他的'书桌整理助手'解决了一个我们每天都会遇到的问题，而且方案非常巧妙"
  • 合影

结束时——关键转化动作：

教练拿着手机走到家长中间：
"各位家长，这就是孩子一天的作品。扫码就能看到孩子的作品集——海报、原型、项目介绍都在里面。转发给家里群、朋友圈，让孩子自己也骄傲一下。"

转化跟进话术（家长单独问时）：

| 家长说 | 教练回应 |

|--------|---------|

| "这个挺有意思" | "今天是一日体验。暑假有一个5天的商科创新营，孩子可以把这个项目真正做深做完整。加个微信我发您详情？" |

| "孩子太小了" | "我们8-16岁都有对应的课程。今天看他的表现，我觉得他很适合继续深入。" |

| "价格多少？" | "今天是一日营¥980。进阶的爱创营是5天¥12,800，能把今天这个原型做成完整的商业计划书。" |

常用话术

开场（09:00）：

"今天你们不是来上课的，是来开公司的。6小时内，发现商机、设计产品、做出原型、上台路演——体验一次当CEO的感觉。"

上午导入（09:30）：

"每个伟大的公司都始于一个问题。Uber觉得打车太麻烦，Airbnb觉得订酒店太贵。你有什么'太麻烦'的时刻？那就是商机。"

下午导入（13:30）：

"上午你们有了产品想法，下午让产品活起来。不用写代码，用AI帮你做——30分钟，一个可以展示的商业原型。"

路演前（15:00）：

"每个创业者最紧张的时刻就是路演前。紧张是正常的——说明你在乎。深呼吸，你有一天的成果在背后，不是空手上去的。"

结束时（16:00）：

"今天你们证明了：用AI创造商业价值，6小时就够。这只是开始。"

模块一：AI认知破冰（30分钟）

1.1 快速体验：AI猜猜看

互动游戏： 以下哪些是AI写的？

A. "请阐述鸦片战争的历史意义和影响。"
B. "这篇文章结构清晰，论据充分，但在第三段的论证逻辑上可以进一步加强。建议补充具体数据支持。"
C. "今天晚上吃什么？"
答案揭晓： 以上三段中，有两段是AI生成的。

| 家长沟通 | 30分钟写通知 | AI生成+个性化调整 |

1.3 中学教师必知的3个AI真相

  • AI不是魔法，是可训练的工具 — 像搜索引擎一样，越会用越有用
  • AI不会替代教师，但会用AI的教师会替代不会用的 — 效率差距可达10倍
  • 教学生用AI，比禁止学生用AI更重要 — AI素养是未来公民的基本素养

模块二：AI备课提速（45分钟）

2.1 极速出题

演示：10秒钟生成一套练习题

打开DeepSeek/文心一言，输入：


你是一位初中[数学/语文/英语...]教师，请为[年级]学生生成
关于[知识点]的5道选择题。

要求：
- 覆盖基础、中等、进阶三个难度
- 每道题附答案和简要解析
- 基础题2道，中等题2道，进阶题1道
💡 关键提示词技巧： 说清楚"你是谁、学生是谁、要什么、什么格式"

进阶：一键生成完整试卷


请为[年级][学科]第[X]单元生成一份单元测试卷：
- 题型分布：选择题10道(30分)、填空题5道(15分)、简答题3道(30分)、
  综合题2道(25分)
- 总分100分，考试时间45分钟
- 附参考答案和评分标准

2.2 极速写教案

传统： 写一份完整教案 → 2-3小时

AI： 生成框架+人工填充 → 20分钟

提示词示例：


你是一位有15年教龄的初中[学科]骨干教师。
请为《[课文/章节名称]》设计一课时（45分钟）教案。

格式要求：
┌──────────────┬──────────────────────────────┐
│ 教学环节      │ 具体内容                      │
├──────────────┼──────────────────────────────┤
│ 教学目标      │ 知识与技能/过程与方法/        │
│              │ 情感态度与价值观               │
├──────────────┼──────────────────────────────┤
│ 教学重难点    │                              │
├──────────────┼──────────────────────────────┤
│ 导入(3分钟)   │                              │
├──────────────┼──────────────────────────────┤
│ 新课讲授(25分)│                              │
├──────────────┼──────────────────────────────┤
│ 课堂练习(10分)│                              │
├──────────────┼──────────────────────────────┤
│ 总结(5分钟)   │                              │
├──────────────┼──────────────────────────────┤
│ 作业布置(2分钟)│                              │
└──────────────┴──────────────────────────────┘

请用表格格式输出，内容要贴合教材，符合新课标要求。

模块三：AI课堂教学（60分钟）

3.1 课堂AI互动三招

第一招：AI即时问答

课堂场景：学生提问超出备课范围


怎么办？→ 当场打开DeepSeek → 输入问题 → 
AI生成回答 → 教师判断+转化后讲解
💡 教师的真正价值不是"知道所有答案"，而是"知道如何找到好答案"

第二招：AI生成课堂案例


请为"光的折射"这个物理概念，生成3个生活中的实例。
要求：初中二年级学生能理解，每个实例附简短的原理说明。

第三招：AI出临场练习题


刚才讲解了"一元二次方程的求根公式"，请出3道即时练习题，
分易、中、难三级，学生能在3分钟内完成。

3.2 AI作业批改助手

操作步骤：

  • 拍摄/扫描学生作业
  • 上传到AI工具（如DeepSeek支持图片上传）
  • 输入提示词："请批改这份[学科]作业，标出错误并给出正确答案和改进建议"
  • 教师审核AI批改结果
⚠️ 重要原则： AI批改是"初筛"，教师要做"终审"。AI标注可能的错误，教师判断是否确实错误以及错误的性质。

# AI智能体研学课 · 40分钟教学方案（初中版）

**适用场景**：公办初中大班授课（40-50人）
**适用年级**：初一至初三
**设计基础**：基于高中版方案降维适配，保留核心框架，降低认知门槛

---

## 一、设计理念

### 核心调整：高中版 → 初中版

| 维度 | 高中版 | 初中版 |
|------|--------|--------|
| 认知负荷 | 高（战略规划、数据分析） | 低（具体任务、可视成果） |
| 课题方向 | 宏大叙事（称霸地球） | 身边话题（校园、兴趣、偶像） |
| AI协作深度 | 多轮追问、迭代优化 | 单轮+简单追问，重在体验 |
| 抽象程度 | 高（战略、体系、方法论） | 低（具体、直观、可触摸） |
| 趣味性 | 成就感驱动 | 好奇心+好玩双驱动 |

### 核心理念不变

> 不是替代老师，而是让老师插上AI的翅膀。

### 初中版特别原则

1. **具体 > 抽象** — 任务要看得见摸得着
2. **好玩 > 有用** — 先让学生觉得有趣，有用自然发生
3. **体验 > 产出** — 质量不重要，重要的是每个人都跟AI互动了
4. **安全感第一** — 确保每个学生知道"怎么提问"，不怕问错

## 二、课前准备清单

### 2.1 技术准备（提前1天）

同高中版：

| 项目 | 说明 |
|------|------|
| 飞书群 × 4个频道 | 每个频道接入AI智能体 |
| 建站模板 × 4 | MiniMax space 空白网站（配色更活泼、可加表情包） |
| 任务卡 × 4 | 打印（字体更大，文字更少，多放图） |
| 计时器 | 40分钟倒计时 |

### 2.2 分组策略

40-50人分 **4个大组**，每组10-12人。

**角色简化**（3种，比高中版少2种）：

```
┌────────────────────────────────────┐
│  👤 提问手（2-3人）                │
│     负责打字问AI，谁想到谁问       │
├────────────────────────────────────┤
│  👤 整理手（3-4人）                │
│     把AI回答里好玩的内容圈出来     │
├────────────────────────────────────┤
│  👤 装修手（3-4人）                │
│     把内容放进网站，调颜色加图     │
└────────────────────────────────────┘
```

> 组长可由老师指定，建议选班里最活跃的同学。

### 2.3 初中版特别准备

| 额外准备 | 说明 |
|----------|------|
| 🃏 **提示词口袋卡** | 每组发一张小卡片，印好3-5个可以直接复制的问题模板（见第六部分） |
| 🎯 **计时音乐** | 每个环节切换时放5秒短音效，制造节奏感 |
| 🏆 **贴纸奖励** | 展示环节每组发一枚贴纸投给"最牛网站" |

## 三、40分钟课堂流程

```
时间轴：
0'───────5'───────10'───────────────────33'──────38'──40'
 │         │         │                     │       │    │
开场热身  分组发卡  自由共创             成果秀  投票 收尾
(3min)    (5min)   (23min)              (5min) (1min)(1min)
```

### ⏱ 0:00-0:03 开场热身（3分钟）

**老师话术参考**：

> "同学们，今天不做题、不听课，我们来玩点真的。
>
> 你们每个人手上有个超级搭档——AI。它能写作文、能画图、能解题、能编故事。
> 但有个条件：你得学会**指挥**它。
>
> 40分钟内，每组要做出一个网站。没错，一个真正的、能打开的网站。
> 
> 先问一个问题：你们觉得AI最擅长什么？"
>
> （等3-5个学生回答）
>
> "好，等会儿你们自己验证。现在倒计时开始——"

**要点**：
- 用"玩"和"指挥"代替"学"和"使用"
- 互动破冰让学生开口，打破陌生感
- 跟高中版的区别：多说"好玩"，少说"创造"

### ⏱ 0:03-0:08 分组 + 发卡（5分钟）

1. 快速分4组（喊学号或座位就近）
2. 每组发：**任务卡1张 + 提示词口袋卡1张**
3. 老师打开建站链接，投到大屏展示空白模板长什么样
4. 学生30秒自选角色

### ⏱ 0:08-0:33 自由共创（25分钟）

**初中版节奏**（比高中版更多节点提醒）：

| 时间段 | 环节 | 老师广播提醒 |
|--------|------|-------------|
| 0:08-0:13 | 讨论+第一次问AI | "先讨论1分钟，然后问AI第一个问题！直接复制口袋卡上的问题就行。" |
| 0:13-0:18 | 看AI回答+继续追问 | "AI回答得怎么样？不满意就继续问：'换个风格'、'再详细点'、'举例子'。" |
| 0:18-0:23 | 深度探索 | "试试让AI换个角度回答，或者让它出个题目考考你们。" |
| 0:23-0:28 | 开始做网站 | "把AI给的内容搬进网站里。先写标题，再贴内容，最后加图片。" |
| 0:28-0:33 | 修整+准备展示 | "最后5分钟！调一下颜色排版，选一个最酷的页面准备展示。" |

**老师巡场引导**（每30秒经过一个组）：

| 场景 | 引导话术 |
|------|---------|
| 学生不知道问什么 | "口袋卡上有问题模板，直接复制就行。" |
| AI回答太长 | "让它总结成3个要点。" |
| 学生觉得不好玩 | "让它给你讲个相关的故事，或者画张图。" |
| 做网站卡住了 | "先填文字，颜色后面再调。能贴上去就行。" |
| 有学生发呆 | "你来负责问AI一个问题——随便什么都可以。" |

### ⏱ 0:33-0:38 成果秀（5分钟）

**每组1分钟**展示：

> "报网址 → 翻到最得意的一页 → 说一个最意外的发现"

**跟高中版的区别**：不强求结构化的"心得"，让学生说"最意外""最好玩"的点。

### ⏱ 0:38-0:39 贴纸投票（1分钟）

每组派代表把一枚贴纸贴在最喜欢的一个网站编号下。

**为什么加这个环节**：初中生需要即时反馈和同伴认可，贴纸投票比口头评价直观得多。

### ⏱ 0:39-0:40 收尾（1分钟）

> "今天你们做了什么？
> 你们指挥了一个超级聪明的AI，40分钟做出了一个网站。
>
> 记住三件事——
> **第一，AI很厉害，但你才是下命令的那个人。**
> **第二，不满意就让它重做，你有这个权力。**
> **第三，下次遇到什么难题，试试问AI——你可能会惊讶。**
>
> 下课！"

## 四、4组课题设计（初中版）

全部换成**初中生熟悉的场景**，降低认知门槛。

### 组一：校园组——"用AI设计理想校园"

| 项目 | 内容 |
|------|------|
| **课题** | 我心目中的未来校园 |
| **任务** | 用AI设计一所你梦想中的学校，可以很夸张 |
| **涉及科目** | 语文（描述）、美术（想象）、信息技术 |
| **AI提示词口袋卡** | ①"帮我设计一所未来学校，说说它长什么样" ②"给这所学校加3个超酷的设施" ③"帮我想一句这所学校的校训" |
| **网站方向** | 未来校园介绍页：校名+设施清单+校训+一张AI配图 |

### 组二：兴趣组——"用AI安利你的最爱"

| 项目 | 内容 |
|------|------|
| **课题** | 我的××安利指南 |
| **任务** | 选一个你最喜欢的（游戏/动漫/运动/明星/电影），用AI整理成安利指南 |
| **涉及科目** | 语文（表达）、美术（排版） |
| **AI提示词口袋卡** | ①"介绍一下××，用3句话让没听说过的人入坑" ②"帮我想5个'入坑理由'" ③"给这个安利页面配一句吸引人的标题" |
| **网站方向** | 安利指南页：入坑理由+必知知识点+推荐清单 |

### 组三：未来组——"用AI想象10年后的世界"

| 项目 | 内容 |
|------|------|
| **课题** | 2036年的某一天 |
| **任务** | 用AI想象10年后的世界，可以从衣食住行里选一个角度 |
| **涉及科目** | 语文（想象作文）、美术、科学 |
| **AI提示词口袋卡** | ①"描述一下2036年普通人的一天" ②"10年后的学校跟现在有什么不同" ③"10年后最酷的3个发明是什么" |
| **网站方向** | 未来世界展示页：3个未来场景+配图 |

### 组四：学霸组——"用AI出份自测题"

| 项目 | 内容 |
|------|------|
| **课题** | XX学科AI趣味自测 |
| **任务** | 选择一门科目，用AI出一套好玩的题目（不是考试那种） |
| **涉及科目** | 数学/英语/语文/科学（自选） |
| **AI提示词口袋卡** | ①"帮我出5道关于××的选择题，带答案" ②"把这5道题改成选择题" ③"加一道『陷阱题』，看谁会掉坑" |
| **网站方向** | 趣味自测页：5-8道题+答案+难度评分 |

## 五、初中版特别注意事项

### 5.1 跟高中版的8个关键区别

| # | 环节 | 高中版做法 | 初中版做法 |
|---|------|-----------|-----------|
| 1 | 开场 | "从课堂到宇宙"宏大叙事 | "来玩点真的"轻松开场 |
| 2 | 任务卡 | 纯文字，半页A4 | 大字+图标+示例，一页A4 |
| 3 | 角色分工 | 5种角色（含发言人） | 3种角色（提问手/整理手/装修手） |
| 4 | 提示词 | 不给模板，鼓励自由发挥 | 给口袋卡，3-5个可直接复制的问题 |
| 5 | 巡场频率 | 每30秒一个组 | 每20秒一个组（更密集） |
| 6 | 展示环节 | 结构化展示（网址+亮点+心得） | 自由展示（最意外/最好玩的点） |
| 7 | 反馈方式 | 老师点评 | 贴纸投票（同伴互评） |
| 8 | 收尾 | "有温度的超级个体" | "你是下命令的那个人" |

### 5.2 常见问题应对

| 问题 | 应对 |
|------|------|
| "老师，AI回答的不对" | "哪里不对？你告诉它，让它重来。指挥AI是你的工作。" |
| "老师，我不会打字" | "让组里打字快的同学帮忙，或者用语音输入。" |
| "我们组不知道选什么" | "口袋卡上4个方向随便选一个，先问AI第一句再说。" |
| "网站做不完" | "没关系，把标题和主要内容放上去就行。40分钟做到这样已经很厉害了。" |
| "有人不参与" | "你负责问一个问题就行——随便什么，问完就完成任务了。" |

### 5.3 特别提醒

- **打字速度**：初中生打字可能不快，建议每组配一个打字快的同学做"提问手"
- **注意力跨度**：每5分钟老师要喊一次进度，保持节奏
- **容错空间**：技术卡顿、登录不上、网站崩了——没关系，让学生体验过程本身
- **不要评价质量**：初中生的成果不需要"评价"，只需要"展示"和"鼓掌"

## 七、初中版课题与高中版对照

| 维度 | 高中版 | 初中版 |
|------|--------|--------|
| 组一 | "AI时代战略"（抽象+宏大） | "未来校园"（具体+身边） |
| 组二 | "学科AI指南"（学科深度） | "安利指南"（兴趣驱动） |
| 组三 | "提分方案"（数据+KPI） | "2036年的某一天"（想象+故事） |
| 组四 | "交互学习工具"（编程+交互） | "趣味自测题"（游戏化+轻量） |

1 课程设计理念

学习目标：理解青少年创造营的设计原则——如何在有限时间内最大化体验感和成就感，实现"从0到1做出作品"的认知闭环。

设计三原则

门槛要低
不需要编程基础，不需要AI经验，不需要自带设备。一切从零开始，确保每个孩子跟上。
成就感要强
6小时必须做出"可展示的作品"。结束后孩子能兴奋地说"今天我做了个AI项目！"
转化要自然
不硬推课程，而让家长看到孩子的变化和潜力。"你的孩子适合继续深造"——由成果说话。

核心变化（v1→v2）

维度v1（AI创客启蒙）v2（AI商科启蒙）
开场问题"AI能做什么？""你想改变什么？"
核心活动AI绘画/音乐创作社会问题探索 → AI海报 → AI游戏
项目产出绘画/音乐作品项目海报 + AI互动游戏 + 路演
结束引导"用AI让世界更好""这是我的项目，我想改变这件事"
课后跟进发送作品图片发送作品集链接+成长建议+暑假营邀请
一句话定位：一天时间，零基础造出人生第一个AI作品。
定价：¥599-899/人（早鸟¥599）

3 教学方法 · 闻思修教学法

学习目标：掌握"闻思修"三环教学法在AI商科课程中的具体应用。

闻·思·修架构

闻
输入 · 激发兴趣
用有趣案例/故事/演示引入概念，抓住注意力。每节课<=15分钟，简洁有力。
思
讨论 · 内化认知
引导讨论+头脑风暴，让学生在对话中理解概念。控制在15分钟内。
修
实践 · 产出成果
动手操作，用AI工具完成具体任务。核心环节，占每节课50%以上时间。
典型的"闻→思→修"一节课结构（60分钟）：
┌─────────────────────────────────────────────────────┐
│ 闻 (15min)    思 (15min)        修 (30min)           │
│ ┌─────────┐  ┌─────────┐  ┌──────────────────────┐  │
│ │ 精彩案例 │→│ 头脑风暴 │→│ 动手实践+AI工具      │  │
│ │ 核心概念 │  │ 小组讨论  │  │ 产出作品             │  │
│ │ 问题提出 │  │ 分享观点  │  │ 导师巡场辅导         │  │
│ └─────────┘  └─────────┘  └──────────────────────┘  │
│                                                      │
│ 讲师为主         学生为主           学生为主+讲师辅导    │
└──────────────────────────────────────────────────────┘
讲师纪律："修"环节讲师不要讲太多！学生动手时，讲师只做巡场辅导。孩子的成就感来自"我自己做出来的"，而不是"老师教得好"。

4 AI工具包 · 一日营必备

学习目标：掌握一日营所需的AI工具清单和使用场景，确保课前准备充分。
启蒙营AI工具：豆包 + 灵光，足够了。
豆包负责AI对话/AI生图/AI音乐，灵光负责AI PPT/AI搜索——两款工具覆盖全部教学场景，无需额外账号。
用途推荐工具课前准备

AI PPT · AI搜索灵光提前测试网页端可用性
设备准备清单：

✓ 备用平板2-3台（部分学生可能没带手机）
✓ 稳定WiFi（支持30+设备同时在线）
✓ 投影/大屏（讲师演示用）
✓ 翻页笔 + 音箱（播放视频/音乐）
✓ 提前准备好豆包App下载二维码 + 灵光网址书签

提示词模板

社会问题探索
"我是一名[年龄]岁的学生。请帮我探索：1）我们身边有哪些社会问题？2）这些问题影响了谁？3）为什么还没有被解决？4）如果我有AI技术，我可以做什么？用简单易懂的语言。"
项目创意激发
"我想做一个关于[主题]的项目。请帮我：1）取一个吸引人的项目名字 2）这个项目帮助谁？解决什么？ 3）我需要什么资源？ 4）怎么让别人支持我？"

5 助教执行手册精华

学习目标：掌握助教在创造营中的角色定位和关键执行要点，确保团队配合无缝。

角色分工

角色人数职责
主讲讲师1人授课、节奏把控、核心环节引导
助教2-3人（1:6配比）巡场辅导、技术支援、纪律维持、个别关注
运营1人家长沟通、拍照/录像、后勤、转化咨询

助教四要四不要

要做 ✓
✓ 学生卡住时，用提问引导而不是直接给答案
✓ 发现优秀作品，立即拍照记录
✓ 关注沉默/不自信的学生，多鼓励
✓ 及时反馈设备/网络问题给运营
不要做 ✗
✗ 不要替学生操作（会破坏成就感）
✗ 不要长时间和单一学生交流（会忽视其他人）
✗ 不要批评学生作品（即使很粗糙）
✗ 不要在课堂上玩手机

关键场景应对

设备/网络故障
运营立即提供备用设备。如果持续故障，让学生2人一组共用一台。不要在课堂上花超过5分钟修设备。
学生不配合/走神
助教走到旁边轻声问"需要帮忙吗？"——90%的情况是卡住了但不敢说。如果持续走神，安排做"课堂记录员"。
特别优秀的学生
提供进阶任务（"你能试试用AI做一个更复杂的功能吗？"）。别让等待同伴的时间变成无聊时间。
家长提前来接
运营主动上前，展示孩子作品，简述今天的成果。这是最好的转化时机——家长看到成果时的表情=成交信号。

7 常见问题（FAQ）

学习目标：掌握家长常见问题的回答策略，建立专业信任。
Q：孩子没接触过编程/AI，能跟上吗？
我们的课程是零基础入门。所有工具都是可视化/自然语言操作——孩子用对话就能用AI。已经有超过500名零基础学员完成课程并做出作品。
Q：一天时间能学到什么？有什么用？
一天的核心不是"学了多少知识"，而是"体验了用AI创造的成就感"。孩子会带走一个完整的AI作品（海报+游戏+作品集），更重要的是——TA会发现"原来我可以用AI做这么酷的事"，点燃持续学习的兴趣。
Q：和外面的编程课/机器人课有什么区别？
传统的AI课教"AI技术是什么"，我们教"用AI创造社会价值"。我们不培养程序员，我们培养未来的创造者。创造思维+AI应用，是其他课程没有的差异化定位。
Q：后续还有什么课程？怎么衔接？
我们设计了完整的进阶路径：一日营（启蒙）→ 暑假爱创营（5天做项目）→ ICC竞赛营（冲刺奖项）→ 背景提升陪跑（长期成果）。您今天看到的孩子在创意上的表现，继续深造会产出可参赛的作品。

讲师/助教课后复盘清单

  1. 今天有多少学生完成了"完整作品"？（目标：90%+）
  2. 哪些环节学生兴趣最高/最低？为什么？
  3. 遇到了哪些设备/网络问题？下次如何预防？
  4. 今天转化了几个暑假营？转化率目标：一日营→暑假营 20%+
  5. 哪些学生值得重点关注？（特别优秀/特别需要帮助）

2 AI编程 · 从零到一

主流AI编程工具对比

工具定位适合场景上手难度
Claude CodeAI编程代理（深度）复杂项目、全栈开发、Agent编排中高
CursorAI原生IDE日常编码、Tab补全、多文件编辑低
WindsurfAI原生IDECascade模式、自动调试修复低
GitHub CopilotAI编程助手代码补全、内联建议低
OpenClawAI编程代理（新颖）开源、终端原生、Python优先中
教学建议：对青少年商科教育，推荐先从Cursor/Windsurf开始（低门槛、有GUI），进阶后引入Claude Code（更高效率、更强Agent能力）。

教育场景应用

AI编程 × 学生
学生可以用自然语言描述一个商业创意的功能，AI直接生成代码原型。不需要会编程，也能做技术验证。适合ICC竞赛中的AI原型搭建。
AI编程 × 教师
教师用AI编程快速制作课件、搭建教学演示平台、自动化作业批改。效率提升10倍以上。

学习目标：理解AI智能体的概念和在商科教育中的应用，掌握无代码智能体搭建的基本方法。

智能体平台对比

平台定位适合特点

7 Paperclip · 零人工公司操作系统

学习目标：了解Paperclip平台的核心理念——什么是"零人工公司"，以及它在商科教育中的启发价值。

什么是Paperclip？

Paperclip是一个开源（MIT协议）的AI Agent编排平台，定位为"零人工公司"的操作系统。它让你可以招募、管理和编排AI智能体团队，在最小人工干预下运营一家"企业"。

旧模式：人 → 指挥 → AI Agent（一对一）

新模式：人 → 设定目标 → Paperclip（公司）→ 管理 → AI Agent 团队（多对多）

零人工公司的组织架构

角色传统公司零人工公司
CEO人类AI Agent
CTO人类AI Agent
工程师人类AI Agent（Claude Code / Codex）
运营人类AI Agent
董事会投资人你（人类）

教育价值

商业思维启蒙：通过"零人工公司"的概念，学生可以理解企业组织架构、角色分工、预算管理、KPI考核等商业基础知识。
升维思考："如果AI可以承担所有执行工作，人的价值在哪里？"——引导学生思考：战略决策、创意方向、价值判断才是人类的核心竞争力。

课堂练习：设计你的"零人工公司"

  1. 选一个你感兴趣的商业创意（如：AI环保助手）
  2. 画出这家公司需要哪些角色（CEO/CTO/运营/市场...）
  3. 哪些角色可以用AI承担？哪些必须由人来完成？
  4. 用AI工具（Claude/Coze）来生成这家公司的"组织架构图"
